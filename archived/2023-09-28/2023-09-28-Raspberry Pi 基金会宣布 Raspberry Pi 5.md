[Raspberry Pi 基金会宣布 Raspberry Pi 5](https://www.solidot.org/story?sid=76221)
------

<div>
Raspberry Pi 基金会宣布了 Raspberry Pi 5，4GB 版本售价 60 美元，8GB 版本 80 美元。主要规格包括：2.4GHz 四核 64位 Arm Cortex-A76 CPU；VideoCore VII GPU，支持 OpenGL ES 3.1 和 Vulkan 1.2；双 4Kp60 HDMI 显示输出；4Kp60 HEVC 解码器；双频段 802.11ac Wi-Fi；蓝牙 5.0 /蓝牙低功耗(BLE)；2 × USB 3.0 端口，2 × USB 2.0 端口，高速 microSD 卡接口，等等。<p></p>
</div>
