[Windows 11 的普及度仍然远落后于 Windows 10](https://www.solidot.org/story?sid=73248)
------

<div>
Statcounter 十月份的统计数据显示，在发布一年之后 Windows 11 的普及度依旧远落后于 Windows 10。目前 Windows 11 的市场占有率为 15.44%，比前一个月增加 1.83%，相比之下 Windows 10 的市场份额为 71.29%，只比前一个月的 71.88% 略下降 0.59%。Windows 7 仍然有 9.61%，Windows 8.1 以 2.45% 位居第四，Windows 8 为 0.69%，Windows XP 为 0.39%。Windows 占据全球桌面操作系统的 76%，其次是 OS X 的 15.7%，Linux 的 2.6%。如果包括移动操作系统，Android 最多占 42.37%、Windows 30.11%、iOS 17.6%、OS X 6.24%、Linux 占 1.04%。
</div>
