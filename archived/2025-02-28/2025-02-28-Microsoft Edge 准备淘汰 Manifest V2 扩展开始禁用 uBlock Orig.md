[Microsoft Edge 准备淘汰 Manifest V2 扩展开始禁用 uBlock Origin](https://www.solidot.org/story?sid=80676)
------

<div>
在 Google Chrome 之后，Microsoft Edge 准备淘汰 Manifest V2 扩展开始禁用 uBlock Origin。最新的 Edge Canary 版本禁用了 Manifest V2 扩展，并显示信息称该扩展不再被支持，Microsoft Edge 建议将其移除。对于被自动禁用的扩展如 uBlock Origin，用户目前暂时还可以在扩展管理中手动启用。<p></p>
</div>
