[粉丝发现 Windows 1.0 彩蛋](https://www.solidot.org/story?sid=71056)
------

<div>
在 Windows 1.0 发布近 37 年之后，狂热粉丝 <a href="https://twitter.com/mswin_bat">Lucas Brooks</a> 发现了一个复活节彩蛋。彩蛋藏在一个笑脸位图文件中，<a href="https://www.pcgamer.com/this-windows-10-easter-egg-managed-to-stay-hidden-for-nearly-37-years/">包含了</a>开发者名单和一个祝贺信息（congratulations）。开发者名单是加密的，提取数据所需要的工具在 Windows 1.0 时代还不存在。在开发者名单中，有一个人在 PC 游戏玩家中非常出名，他就是 Valve 联合创始人 Gabe Newell。Gabe Newell 是在哈佛辍学后在微软开始职业生涯，参与开发了 Windows 系统的前三个版本，领导了 Doom 的 Windows 移植。
</div>
