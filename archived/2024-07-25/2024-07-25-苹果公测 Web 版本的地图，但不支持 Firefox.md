[苹果公测 Web 版本的地图，但不支持 Firefox](https://www.solidot.org/story?sid=78803)
------

<div>
苹果开始公测其地图服务的 Web 版本，目前该版本不支持 Firefox 浏览器。苹果称，用户可通过地图获取驾车和步行路线，寻找地点和有用信息如括照片、营业时间、评分和评论，可以直接从地图的位置卡执行订餐等操作，等等。部分功能如 Look Around 将在未来几个月推出。Web 地图目前只提供英文版，支持 Mac 和 iPad 上的 Safari 和 Chrome 浏览器，以及 Windows PC 上的 Chrome 和 Edge 浏览器。它将在未来支持其他语言、浏览器和平台，但没有给出具体时间。<p></p>
</div>
