[Google 开始向所有 Chrome 用户推送隐私沙盒 API](https://www.solidot.org/story?sid=75577)
------

<div>
随着 Chrome 115 的发布，Google 宣布开始向所有 Chrome 用户启用隐私沙盒相关 API。隐私沙盒（Privacy Sandbox）将取代第三方 Cookie，它跟踪用户的兴趣，允许广告商根据兴趣展示广告，而用户可以管理兴趣并对其归类分组。隐私沙盒代表着广告行业的一次巨大变化。Chrome 115 标志着隐私沙盒相关的 API GA（普遍可用），开发者可对其进行测试。Google 的目标是到 8 月底对 99% 的用户启用隐私沙盒相关 API，第三方 Cookie 将会继续保留一段时间。从 2024 年开始，Google 将对 1% 的 Chrome 用户禁用第三方 Cookie，然而到 2024 年底完全淘汰第三方 Cookie。<p></p>
</div>
