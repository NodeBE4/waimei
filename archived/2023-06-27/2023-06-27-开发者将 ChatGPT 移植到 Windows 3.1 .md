[开发者将 ChatGPT 移植到 Windows 3.1](https://www.solidot.org/story?sid=75354)
------

<div>
微软正在 Windows 11 中整合 ChatGPT 的技术，它没有计划对其它版本的 Windows 做类似的工作。但如果你用  Windows 3.1 或 Windows 95 PC 的话，你可以松一口气了。有开发者将 ChatGPT 客户端移植到了 Windows 3.1。该版本被称为 WinGPT，支持 16 位或 32 位的 Windows 3.1 及更新的操作系统，但不原生支持 64 位系统。 能在三十年前的操作系统上运行 WinGPT 是因为绝大部分处理工作都在 OpenAI 的服务器而不是本地完成。<p></p>
</div>
