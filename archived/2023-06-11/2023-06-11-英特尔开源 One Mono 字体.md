[英特尔开源 One Mono 字体](https://www.solidot.org/story?sid=75216)
------

<div>
英特尔开源了一种为程序员设计的等宽字体 One Mono，采用 SIL Open Font License 1.1 许可证托管在 GitHub 上。该字体由 Frere-Jones Type 联合 Intel Brand Team 和 VMLY&amp;R 设计，特别考虑了弱视开发者的需求，最大程度的提高可识别性，解决开发者的疲乏和眼睛疲劳问题，减少编程错误。在字体设计的每一个阶段，弱视和法定盲人开发者都提供了反馈。Intel One Mono 还涵盖了逾 200 种使用拉丁文字的语言。Intel One Mono 字体提供了四种粗细——细型、常规、中等和粗体——以及相应的匹配斜体。<p></p>
</div>
