[自由软件是时候放弃 GitHub](https://www.solidot.org/story?sid=72007)
------

<div>
GitHub 刚刚宣布<a href="https://www.solidot.org/story?sid=71902">商业化</a>其 AI 编程助手 Copilot。Copilot 是用开源自由软件代码库训练的，在开发者写代码时根据函数名等上下文自动补充完后续代码，很多时候 Copilot 补充的是开源代码库中的代码拷贝。这就引起了许可证方面的问题。致力于推广和捍卫开源自由软件的非盈利组织 Software Freedom Conservancy(SFC) 尝试与微软/GitHub 讨论解决这个问题，但一再遭到漠视甚至无视。它正式发表声明，<a href="https://sfconservancy.org/blog/2022/jun/30/give-up-github-launch/" target="_blank">宣布将致力于帮助开源自由软件项目迁移出 GitHub</a>，对现有成员项目不强制，但不再接受没有从 GitHub 迁移出去的长期计划的新成员。SFC 称它曾询问微软/GitHub 为什么只用开源自由软件训练 Copilot，而不将私有的 Microsoft Windows 和 Office 代码库作为训练集的一部分？微软拒绝回答，这暗示了它更注重自己的“知识产权”，乐于忽视和侵蚀开源自由软件用户的权利。
</div>
