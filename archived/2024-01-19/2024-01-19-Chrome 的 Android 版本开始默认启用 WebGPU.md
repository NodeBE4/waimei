[Chrome 的 Android 版本开始默认启用 WebGPU](https://www.solidot.org/story?sid=77172)
------

<div>
Google Chrome 团队宣布，Chrome 121 在运行 Android 12 或更高版本以及高通 GPU 和 ARM GPU 提供支持的设备上默认启用 WebGPU。Google 计划逐步扩大支持范围，以涵盖更广泛的 Android 设备，包括运行 Android 11 的设备。WebGPU 为 Web 提供了下一代高性能图形 API，允许在 GPU 上执行渲染和计算操作，该 API 此前只限于支持 Direct3D 12 的 Windows PC、支持 Vulkan 的 macOS 和 ChromeOS 设备。Google 称，WebGPU 大幅减少了相同图形的 JavaScript 工作负载，机器学习模型推理性能改进了三倍以上。<p></p>
</div>
