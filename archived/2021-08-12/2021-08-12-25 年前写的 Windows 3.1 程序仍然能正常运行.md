[25 年前写的 Windows 3.1 程序仍然能正常运行](https://www.solidot.org/story?sid=68523)
------

<div>
悉尼大学的 Mark Pesce 在准备 SIGGRAPH 演讲时翻出了 20-30 年前写的程序，其中包括用于浏览 3D 内容的浏览器 <a href="https://markpesce.com/labyrinth" target="_blank">Labyrinth</a>。<a href="https://en.wikipedia.org/wiki/Mark_Pesce">Mark Pesce </a>是一位未来学家，在 1991 年创办了以 William Gibson 小说《神经漫游者》中虚构公司名字命名的 Ono-Sendai Corporation，它是最早一批虚拟现实创业公司。他尝试解压和运行这个为 Windows 3.1 操作系统编写的程序，结果惊讶的发现它在今天仍然能正常运行，甚至还能成功加载为 3D Web 开发的 <a href="http://hyperreal.org/~mpesce/banana.wrl" target="_blank">Cyberbanana</a>。他<a href="https://www.theregister.com/2021/08/11/column_cyberbanana_windows/">称赞了</a>微软在操作系统向后兼容方面做出的成就。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
