[又一家外企撤出中国 思杰母公司将停在华交易](https://www.epochtimes.com/gb/23/11/9/n14112568.htm)
------

<div>
<p>【大纪元2023年11月09日讯】（大纪元记者萧律生报导）近日，<a href="https://www.epochtimes.com/gb/tag/%E7%BE%8E%E5%9B%BD.html">美国</a><a href="https://www.epochtimes.com/gb/tag/%E6%80%9D%E6%9D%B0%E7%B3%BB%E7%BB%9F.html">思杰系统</a>（Citrix）母公司Cloud Software Group决定自今年12月3日起，停止在中国的所有新商业交易。这是最新一家撤出在华业务的<a href="https://www.epochtimes.com/gb/tag/%E7%BE%8E%E5%9B%BD.html">美国</a><a href="https://www.epochtimes.com/gb/tag/%E7%A7%91%E6%8A%80%E5%85%AC%E5%8F%B8.html">科技公司</a>。</p><p>日前，网传一份邮件显示，美国<a href="https://www.epochtimes.com/gb/tag/%E6%80%9D%E6%9D%B0%E7%B3%BB%E7%BB%9F.html">思杰系统</a>（Citrix System）的母公司Cloud Software Group以业务成本上涨为由，宣布退出中国市场，决定于12月3日生效。</p><p>具体为停止在中国大陆、香港和澳门，与客户直接或通过渠道合作伙伴进行的所有新商业交易，但会继续为签订合同的客户提供维护和支持服务。</p><figure id="attachment_14112622" aria-describedby="caption-attachment-14112622" style="width: 450px" class="wp-caption aligncenter"><a target="_blank" href="https://i.epochtimes.com/assets/uploads/2023/11/id14112622-11.jpeg"><img class=" wp-image-14112622" src="https://i.epochtimes.com/assets/uploads/2023/11/id14112622-11-600x688.jpeg" alt="" width="450" height="516" /></a><figcaption id="caption-attachment-14112622" class="wp-caption-text">2023年11月，网传一份邮件显示，美国思杰系统（Citrix System）的母公司Cloud Software Group以业务成本上涨为由，宣布退出中国市场。（网页截图）</figcaption></figure><p>Cloud Software Group的一名发言人向《华尔街日报》证实该邮件内容真实性，并表示该决定将影响到旗下所有公司。</p><p>9日的报导称，Cloud Software Group此举让其成为最新一家<a href="https://www.epochtimes.com/gb/tag/%E6%92%A4%E5%87%BA%E4%B8%AD%E5%9B%BD.html">撤出中国</a>市场的美国<a href="https://www.epochtimes.com/gb/tag/%E7%A7%91%E6%8A%80%E5%85%AC%E5%8F%B8.html">科技公司</a>。</p><p>云计算软件集团Cloud Software Group总部位于佛罗里达州，拥有企业软件供应商思杰（Citrix）、Tibco Softwar等。</p><p>思杰曾在纳斯达克上市，后来并入Cloud Software Group。该公司网站显示，在北京、上海、广州和香港设有销售办事处。思杰曾为京东（JD.com）、中共国有汽车制造商上海汽车集团（SAIC）等中国大型公司提供服务。</p><p>责任编辑：李沐恩#</p>
