[思杰所有者成最新一家撤出中国的美国公司](https://www.zaobao.com/realtime/china/story20231109-1448872)
------

<div>
<p>拥有企业软体品牌思杰（Citri）的Cloud Software Group将停止在中国的业务交易，成为最新一家撤出中国市场的美国公司。</p><p>《华尔街日报》看到Cloud Software Group在星期一（11月6日）发给客户和合作伙伴的电邮中说，已决定于12月3日在中国停止所有新的商业交易，原因是市场成本上升。</p><p>该公司说，将继续为与其签订合同的客户提供维护和支持服务，并对服务中断表示歉意。</p><section id="imu"><div id="dfp-ad-imu1"></div></section><p>Cloud Software Group的发言人证实了邮件内容，并称这一决定影响到其旗下的所有公司。</p><p>总部位于美国佛罗里达州的Cloud Software Group，旗下包括思杰和Tibco在内的企业软体供应商。思杰生产办公室软体，允许用户在线访问桌面以及其他云计算功能。该公司向企业提供软体解决方案，并与渠道合作伙伴合作，为终端客户提供服务。</p><p>随着中国经济增长放缓，以及对国家安全和数据相关问题的担忧加剧，Cloud Software Group是最新一家撤出或大幅收缩在华业务的美国科技公司。</p><div id="innity-in-post"></div><div id="dfp-ad-midarticlespecial"></div><p>思杰曾在纳斯达克上市，2022年被Vista Equity Partners和Elliott Management的一个分支机构私有化，后来并入Cloud Software Group。该公司网站显示，在北京、上海、广州和香港设有销售办事处。</p><p>思杰曾为中国大型企业提供服务，包括电子商务公司京东和国有汽车制造商上海汽车集团。</p><div class="cx_paywall_placeholder" id="sph_cdp_40"></div>
