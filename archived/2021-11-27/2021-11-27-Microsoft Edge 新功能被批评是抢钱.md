[Microsoft Edge 新功能被批评是抢钱](https://www.solidot.org/story?sid=69788)
------

<div>
Microsoft Edge 正在测试一项新功能，允许用户在线购物时分期付款。该功能被称为 Buy now, pay later (BNPL)，由微软与 Zip 合作提供，该分期付款功能正在 Microsoft Edge Dev 和 Canary 渠道测试。购买金额的范围在 $35-$1,000，可分成四次在六周内付清。BNPL 功能<a href="https://it.slashdot.org/story/21/11/26/1843243/microsoft-edges-latest-feature-called-a-shameless-cash-grab-by-critics" target="_blank">遭到了微软粉丝的强烈批评</a>。微软用户主要批评 Edge 团队没有专注于改进浏览器体验本身，反而添加了各种臃肿的功能，认为这个可选的功能是“无耻抢钱”。
</div>
