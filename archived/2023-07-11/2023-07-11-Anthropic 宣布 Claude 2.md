[Anthropic 宣布 Claude 2](https://www.solidot.org/story?sid=75485)
------

<div>
AI 创业公司 Anthropic 宣布了新大语言模型 Claude 2。Claude 2 改进了性能，响应时间更长，可通过 API 访问，它同时推出了面向公众的 beta 网站 claude.ai（尚未对中国大陆开放）。Anthropic 称，Claude 2 在律考多项选择部分的得分为 76.5%，高于 Claude 1.3 的 73.0%；相比申请研究生院的大学生，Claude 2 在 GRE 阅读和写作考试中的得分高于 90%，数量推理得分与申请者的中位数相当。Anthropic 表示，Claude 2 的企业 API 价格与 Claude 1.3 相同。<p></p>
