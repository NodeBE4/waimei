[只有 Windows 11 专业版才能使用本地账户](https://www.solidot.org/story?sid=68204)
------

<div>
安装 Windows 11 家庭版<a href="https://www.pcworld.com/article/3623830/only-windows-11-pro-will-let-you-install-windows-11-with-a-local-account.html">将需要 Microsoft 账户 (MSA)</a>，而安装时本地账户仅提供给 Windows 11 专业版用户。购买预装 Windows 11 Pro PC 的用户可以在安装时选择本地账户或 MSA。家庭版在开始安装时必须使用 MSA，但在安装完成之后可以选择转移到本地账户。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
