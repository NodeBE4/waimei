[在本地运行的 Emacs 辅助编程助手 Emacs Copilot](https://www.solidot.org/story?sid=77017)
------

<div>
今天流行的 IDE 都配备了辅助编程工具如 Github Copilot，但这些工具都运行在云端而非本地。开源 IDE 的辅助编程工具需要能自由运行在本地。开发者 Justine Tunney 发布了在本地运行的 Emacs 辅助编程助手 Emacs Copilot。微软的很多 AI 工具都命名为 Copilot（如 Github Copilot, MS Copilot 和 Office365 Copilot），但 Copilot 并不是微软的商标，软件巨人目前申请的商标是 MICROSOFT COPILOT，尚未获得批准。本地运行的大模型对计算机性能有要求，如果使用 WizardCoder 34b，需要 Mac Studio M2 Ultra 或更高型号，普通 PC 可能需要使用性能略低的 WizardCoder-Python-13b。<p></p>
</div>
