[Threads 准备推出 Web 版本](https://www.solidot.org/story?sid=75851)
------

<div>
Meta 的微博客应用 Threads 准备推出 Web 版本，此前用户可以在桌面浏览 Threads 上的帖子但无法执行其它功能。Web 版本将允许用户像 Twitter/X 的桌面版本那样发帖或关注。Meta 正在内部对 Threads 进行测试，计划是在本周内推出，但并不确定。Threads 用户已经看到部分帖子标记是来自 Web 端而不是应用。<p></p>
</div>
