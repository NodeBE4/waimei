[收到 DMCA 通知后 GitHub 去年移除逾 2 万个项目](https://www.solidot.org/story?sid=77577)
------

<div>
GitHub 更新了它的透明度报告，披露了 2023 年的数据：收到了逾 2,000 DMCA 通知，有 20,517 个项目受到影响。其中只有 35 个 DMCA 通知有争议或被撤回，结果是有 65 个受影响项目没有移除。GitHub 是最大的代码托管平台，托管了逾 4.2 亿代码库，难免会存在有版权问题的项目。比如未经作者授权使用了代码，或者将 GitHub 变成存储盗版电子书甚至音乐的免费库，或者部分项目是被认为是盗版工具或应用。过去几年最受瞩目的 DMCA 通知是 RIAA 对 YouTube-DL 项目发出了移除通知，该项目最终还是恢复了。<p></p>
</div>
