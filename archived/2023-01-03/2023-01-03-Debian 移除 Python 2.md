[Debian 移除 Python 2](https://www.solidot.org/story?sid=73791)
------

<div>
Debian 从 FTP 服务器上移除了 Python 2 包。Python 2 已经在 2020 年结束支持，Python 开发者在 2020 年 4 月发布了 Python 2.7 分支的最后一个版本也是一个非常特殊的版本：Python 2.7.18（近似欧拉常数）。大部分 Linux 发行版开始着手移除  Python 2 包。<p></p>
