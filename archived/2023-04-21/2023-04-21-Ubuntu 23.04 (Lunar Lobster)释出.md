[Ubuntu 23.04 (Lunar Lobster)释出](https://www.solidot.org/story?sid=74738)
------

<div>
Canonical 释出了代号为 Lunar Lobster 的 Ubuntu 23.04。主要新特性包括：新的安装程序，统一了服务器版和桌面版的安装引擎，UI 使用了全新的用户界面；GNOME 4；Ubuntu Desktop 成为首个支持 Azure Active Directory 用户认证的 Linux 桌面；Python、Java、Go、C、C++、Rust 等语言的最新工具链和运行时，支持 armhf、arm64、RISC-V 和 s390x 等不同 CPU 架构；Firefox 111，LibreOffice 7.5.2，支持 RISC-V，Thunderbird 102.10，等等。<p></p>
</div>
