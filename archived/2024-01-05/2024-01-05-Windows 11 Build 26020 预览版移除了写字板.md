[Windows 11 Build 26020 预览版移除了写字板](https://www.solidot.org/story?sid=77064)
------

<div>
微软本周释出了最新的预览版本 Windows 11 Build 26020，正式杀死了写字板应用。写字板应用最早是随 Windows 95 一起发布的，是一种简单的文字处理器，它的功能强于记事本但弱于 Microsoft Word。微软在去年 9 月宣布将在未来版本中移除写字板，推荐用户将 Microsoft Word 用于富文本文件如.doc 和.rtf 的处理，将记事本用于纯文本文件如 .txt 的处理。Windows 11 Build 26020 除了杀死写字板，还杀死了没什么人使用的通讯录应用 People 和步骤记录器（Steps Recorder）。<p></p>
</div>
