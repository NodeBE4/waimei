[Steam 停止支持 Windows 7 和 Windows 8](https://www.solidot.org/story?sid=79774)
------

<div>
Steam 客户端最新更新停止了对 Windows 7 和 Windows 8 的支持。此举并不出人意料，微软早已停止支持这些旧版本 Windows，而 Valve 提前了两年向客户通知了它的计划。Steam 现在只支持 Windows 10、Windows 11、Linux 和 macOS。根据 10 月的 Steam 硬件调查，只有 0.28% 的 Windows 玩家运行 Windows 7，而运行 Windows 8 和 Windows 8.1 的玩家人数则更少。<p></p>
</div>
