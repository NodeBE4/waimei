[亚马逊 AWS 将向印度投资 127 亿美元](https://www.solidot.org/story?sid=75001)
------

<div>
亚马逊计划在 2030 年前向印度的云计算业务投资 127 亿美元，扩大 AWS 基础设施在这一关键海外市场的规模。亚马逊此前在印度投资 37 亿美元建设 AWS 基础设施，目前在当地有两大数据中心区域。电商巨头表示它的投资将能支持 13.17 万全职工作，横跨工程、电信和建筑等领域。印度是全球第二大互联网市场，亚马逊 AWS 是印度最大的云服务商，其客户包括了印度电子和信息技术部和 Axis Bank 等。Google 在印度也有两个数据中心区域，而微软有三个，它们都计划扩大在印度的云服务。<p></p>
</div>
