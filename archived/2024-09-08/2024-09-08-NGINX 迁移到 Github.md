[NGINX 迁移到 Github](https://www.solidot.org/story?sid=79193)
------

<div>
NGINX 项目宣布从 Mercurial 迁移到 Github。从现在开始，NGINX 项目将以 Pull Requests 形式接受贡献，通过 Github 问题页接受 bug 报告、功能请求和功能增强建议，将 GitHub 的讨论页面成为社区论坛。NGINX 项目表示将给予开发者们过渡时间，在 2024 年 12 月 31 日前继续通过邮件列表接受补丁和社区支持。<p></p>
</div>
