[Mozilla 和 Open Web Docs 在 MDN 上携手合作](https://www.solidot.org/story?sid=70992)
------

<div>
Mozilla 和 Open Web Docs 宣布在 MDN 项目的  Web Docs 部分<a href="https://hacks.mozilla.org/2022/03/mozilla-and-open-web-docs-working-together-on-mdn/">携手合作</a>。Mozilla 宣布将推出订阅服务 MDN Plus，而 Web Docs 部分将会永远免费：Mozilla 承担平台的基础设施、开发和维护费用，包括一个工程师团队和一个专业技术作者团队；Open Web Docs 接受企业和个人的捐款，捐款不会转给 Mozilla 或 MDN，而是提供给 MDN  Web Docs 的文档技术写作团队。
</div>
