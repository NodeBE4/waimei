[ProtonMail 从网站上删除不记录用户 IP 的声明](https://www.solidot.org/story?sid=68845)
------

<div>
加密邮箱服务 ProtonMail 因遵守其所在地瑞士的命令而记录法国活动人士的邮箱 IP 地址引发了广泛争议。它随后被发现从网站上<a href="https://www.theregister.com/2021/09/07/protonmail_hands_user_ip_address_police/" target="_blank">删除了</a>不记录用户 IP 的声明。此前 ProtonMail 在网站上声明创建邮箱账号无需任何个人信息，默认情况下它不记录会关联到匿名邮箱地址的任何 IP 日志，因为用户隐私至上。现在，ProtonMail 将声明修改为它尊重用户隐私将用户放在第一位，用户的数据仍然属于用户，加密可以保证做到这一点。ProtonMail 同时更新了隐私政策，声明如果用户违反了瑞士法律，作为犯罪调查的一部分，它有法律义务记录用户 IP 地址。
</div>
