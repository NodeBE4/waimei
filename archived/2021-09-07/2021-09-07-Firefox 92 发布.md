[Firefox 92 发布](https://www.solidot.org/story?sid=68844)
------

<div>
Mozilla 释出了 <a href="https://www.mozilla.org/en-US/firefox/92.0/releasenotes/">Firefox 92</a>。主要新特性包括：使用 HTTPS RR 自动升级到 HTTPS，其中 RR 代表 <a href="https://emilymstark.com/2020/10/24/strict-transport-security-vs-https-resource-records-the-showdown.html">Resource Records</a>，是一种新型的 DNS 记录，本质上说如果一个域名支持 HTTPS 那么在浏览器上输入该域名的 HTTP 地址将会自动升级到 HTTPS；全域色彩级支持视频播放；Mac 用户可通过浏览器文件菜单访问 macOS 共享选项；macOS 支持 ICC v4 图像。此外还有 bug 和安全修正等等。
</div>
