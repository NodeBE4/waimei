[Linux 5.20 改名为 Linux 6.0](https://www.solidot.org/story?sid=72328)
------

<div>
在 Linux 5.19 释出之后，下一个版本叫 Linux 5.20 还是 Linux 6.0，经常被大数字搞混的 Linus Torvalds 表示下一个版本将被称为 6.0。这一做法和之前 Linux 3.19 的下个版本是 Linux 4.0 以及 Linux 4.19 的下一个版本是 Linux 5.0 类似。根据内核邮件列表 <a href="https://lore.kernel.org/lkml/?t=20220801195600">pull request</a>，Linux 6.0 基本是已经普遍接受。根据这些 pull request，Linux 6.0 的新变化<a href="https://www.phoronix.com/news/Linux-6.0-Scheduler" target="_blank">将包括</a>：改进 AMD Zen 系统的 NUMA 平衡，在高系统负载下更有效的 CPU 闲置搜索，移除 DECnet 网络代码，等等。
</div>
