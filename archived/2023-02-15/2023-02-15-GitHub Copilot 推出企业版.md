[GitHub Copilot 推出企业版](https://www.solidot.org/story?sid=74137)
------

<div>
GitHub 的 AI 编程助手 Copilot 推出了企业版。 GitHub 称，Copilot 帮助逾百万开发者提高了生产力，开发者的编程速度最高快了 55%。企业版 Copilot for Business 的优势在于：AI 模型更强大，能自动过滤掉不安全的代码建议，支持 VPN 代理，简化登陆。GitHub 表示，今天所有编程语言的代码有 46% 使用 Copilot 帮助构建，其中 Java 开发者使用 Copilot 的比例高达 61%。<p></p>
</div>
