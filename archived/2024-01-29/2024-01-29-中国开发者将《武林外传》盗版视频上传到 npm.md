[中国开发者将《武林外传》盗版视频上传到 npm](https://www.solidot.org/story?sid=77251)
------

<div>
在将 GitHub 和 npm 作为电子书存储库之后，又有中国开发者将 npm 作为盗版视频存储库，npm 母公司 GitHub 已将这些内容移除。GitHub 和 npm 主要被用于共享软件代码。上传到 npmjs.com 的视频是流行情景喜剧《武林外传》，总共分成 748 个包，每个包大小大约 54.5 MB，解压后的视频后缀为.ts，代表它们是从 DVD 和蓝光光盘转录而来。它们由名叫 wlwz 的用户于去年 12 月 4 日上传，视频使用的名字是 wlwz-2312-1405 之类。这位用户的账号已经删除。<p></p>
</div>
