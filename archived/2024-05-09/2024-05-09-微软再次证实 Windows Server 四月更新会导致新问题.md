[微软再次证实 Windows Server 四月更新会导致新问题](https://www.solidot.org/story?sid=78124)
------

<div>
微软证实 Windows Server 四月更新可能会导致 Local Security Authority Subsystem Service (LSASS)进程崩溃，域控制器重启的问题。LSASS 服务被用于处理安全策略、用户登录、访问令牌创建和密码更改。受影响的 Windows 版本和有问题的安全更新包括 Windows Server 2022 (KB5036909)、Windows Server 2019 (KB5036896)、Windows Server 2016 (KB5036899)、Windows Server 2012 R2 (KB5036960)、Windows Server 2012 (KB5036969)、Windows Server 2008 R2 (KB5036967) 和 Windows Server 2008 (KB5036932)。微软的四月安全更新已知会导致 NTLM 身份认证失败和 VPN 连接问题。<p></p>
</div>
