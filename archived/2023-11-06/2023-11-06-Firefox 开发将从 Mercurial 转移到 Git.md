[Firefox 开发将从 Mercurial 转移到 Git](https://www.solidot.org/story?sid=76542)
------

<div>
Firefox 桌面开发长期以来一直支持 Mercurial 和 Git 两种版本控制系统，这给团队带来了巨大负担，因此 Mozilla 决定将开发转移到 Git。Firefox 开发团队将会继续使用 Bugzilla、moz-phab、Phabricator 和 Lando，虽然项目将会托管在 GitHub，但短时间内不会接受 Pull Requests，贡献工作流维持不变。目前还是计划阶段，开始迁移至少要等到六个月之后。<p></p>
</div>
