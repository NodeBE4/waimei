Soros）今天稱如果俄羅斯輸掉烏克蘭戰爭，他口中的「俄羅斯帝國」將會解體，前蘇聯加盟共和國都將樂見其成。</p><p>路透社報導，俄羅斯去年2月24日入侵烏克蘭，引發二戰以來最致命的歐洲衝突，也是1962年古巴飛彈危機以來，俄羅斯與西方之間最大的衝突。</p><div class='media'><div class='insertGroup'><div><a class='insert' 俄軍被爆2月每天逾800人喪命</span></a></div></div></div><p>索羅斯指出，美國會支持烏克蘭，但美國總統拜登（Joe Biden）已提醒烏克蘭總統澤倫斯基（Volodymyr Zelenskyy），美方的支持將有限度，必須避免引發第三次世界大戰。索羅斯並未透露他是自何處獲得上述訊息。</p><p>從避險基金管理人轉為慈善家的索羅斯表示，俄羅斯軍隊領導不善、裝備落後且士氣低落，但俄羅斯總統蒲亭（Vladimir Putin）為了擊敗烏克蘭軍隊，轉而求助傭兵組織瓦格納集團（Wagner Group）。</p><div class='dictionary'><span>根據索羅斯辦公室發布的演講稿，索羅斯在</span><span class='news-dic-inline' data-id='001400'><span>慕尼黑安全會議</span><span class='dic-icon'></span></span><div class='dictionary-box' data-box='001400'><p class='dictionary-title'>慕尼黑安全會議</p><div class='news-dic-text'><p>慕尼黑安全會議（Munich Security Conference, MSC）於1963年首度召開，是有關國際安全政策的年度會議，前身是國際防禦科學會議（Internationale Wehrkunde-Begegnung），為國際上規模最大的同類安全事務會議。</p><p>在過去數十年，慕尼黑安全會議已成為討論國際安全政策最為重要的獨立論壇。每年都有來自世界各國數百位高層人士參加會議，就當前和未來的安全挑戰展開激烈辯論。</p><p>與會者包括國家元首、部會首長、國際和非政府組織的重要人物、產業界、媒體業、學術界和民間社會的高級代表。</p><p>會議每年2月召開，舉辦地點在德國慕尼黑的巴伐利亞霍夫飯店（Hotel Bayerischer Hof）。</p></div><div class='news-dic-all seeArticlemore'>看更多</div><div class='news-dic-all closeBox'>關閉</div></div><span>（Munich Security Conference）上說，「前蘇聯加盟共和國迫不及待地想看到俄羅斯在烏克蘭受挫，因為他們想維護本身獨立。」</span></div><p>索羅斯表示，「這代表烏克蘭獲勝將導致俄羅斯帝國解體，俄羅斯將不再對歐洲與世界形成威脅。」「那將是一個重大但更好的改變。」</p><p>核武超級強權蘇聯1991年解體成15個獨立共和國，但俄羅斯仍然是其中最強大的國家，擁有世上最大的核武庫。</p><p>蒲亭認為西方富野心且傲慢，他將烏克蘭戰爭描述為與西方間的生死存亡之戰，並表示俄羅斯將用一切可能手段來保護自己及其人民免受攻擊。</p><p>索羅斯表示，蒲亭押寶普里格津（Yevgeny Prigozhin）成立的瓦格納集團，短期內似乎奏效。</p><p>索羅斯說俄羅斯「押對寶了」，「烏克蘭面臨一個戰略選擇：要不然就是為阻止瓦格納集團而愈陷愈深，要不然就是向俄羅斯奉上宣傳勝利，保留本身有限資源以待反擊」。</p><p>索羅斯指出，烏克蘭一旦在春天得到西方承諾的武器，會有一些渺小的機會。</p><p>關於中國，索羅斯表示，習近平主席突然放棄「清零政策」，導致外界對共產黨的信任產生動搖。</p><p>索羅斯說，中國「目前的形勢滿足了政權更替或革命的所有先決條件」「但這只是一個不透明過程的開始，一段時間後才會感受其影響」。</p><p>索羅斯說：「短期內習近平可望持續掌權，因為他牢牢控制所有鎮壓工具。」「但我相信習近平不會一輩子掌權，而在他當政期間，中國也不會成為習近平所追求的軍事與政治主宰者。」</p><p>以國土面積而言，俄羅斯是世界上最大的國家；中國則是世界第二大經濟體。索羅斯至今視俄羅斯與中國為一群興起中的「封閉社會」主要成員，在這些社會中，個人服從於國家。（譯者：屈享平/核稿：嚴思祺）1120217</p><br><hr><div>获取更多RSS：<br><a
