[Google 将举办最后一届 Code Jam 编程竞赛](https://www.solidot.org/story?sid=74210)
------

<div>
Google Code Jam 国际编程竞赛将在举办二十年后结束，今年 4 月 15 日的比赛将是最后一届，采用线上比赛，自 2020 年以来，因新冠疫情比赛改为线上举办。Code Jam 始于 2003 年，比赛内容包括一系列的算法问题，参赛者必须在指定时间内解决。参赛者允许使用任意自选编程语言和开发环境来解答问题，奖金最高为 1 万美元。过去二十年，中国的楼天城（两次）和白俄的 Gennady Korotkevich（八次） 多次获得了冠军，按国家统计，白俄最多赢得了 9 次冠军，中国第二 3 次，俄罗斯 2 次，波兰 2 次。<p></p>
</div>
