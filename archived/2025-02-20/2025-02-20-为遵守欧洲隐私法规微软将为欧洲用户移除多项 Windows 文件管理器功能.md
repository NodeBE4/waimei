[为遵守欧洲隐私法规微软将为欧洲用户移除多项 Windows 文件管理器功能](https://www.solidot.org/story?sid=80598)
------

<div>
为遵守欧洲隐私法规，微软将为欧洲经济区用户移除多项 Windows 文件管理器功能。微软向 Windows Insider 测试者释出了一个预览版本 Windows 11 24H2 build 26120.3281。其中一个变化是移除 Location History API，该 API 允许 Windows 语音助手 Cortana 在位置启用的情况下访问 24 小时位置历史。Cortana 已被弃用，Windows 的助手现在都变成了 Copilot。另一个变化适用于欧洲经济区 Entra ID 账号。为满足欧盟隐私规定，微软从 Windows 文件管理器中移除了跟踪用户活动的功能，包括 Recent（最近访问的文件）、Favorites（收藏夹）、Details Pane（详细信息窗格）和 Recommended（推荐内容）。欧盟用户的文件管理器将会更干净整洁，可能性能也会有所提升。<p></p>
</div>
