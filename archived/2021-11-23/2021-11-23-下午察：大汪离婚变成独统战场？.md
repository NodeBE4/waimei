<p>台湾艺人大S和大陆富二代汪小菲真的官宣离婚了。</p><p>这对半年前就传出婚姻触礁的明星夫妻，其虚虚实实的纷争在经过各路人马戏剧化的演绎后，昨天正式为彼此画上句点。</p><p>明星婚姻向来不是个人私事，大S和汪小菲的离婚也不例外。谁出轨？谁有小三？家产怎么分？孩子归谁？孰是孰非？这类话题往往伴随着离婚消息在网上发酵。</p><section id="imu"><div id="dfp-ad-imu1">        </div></section><p>不过，本届吃瓜群众的“格局”要比以往更高，他们关心的不是往常那些琐碎的情感问题，而是将这场婚姻的破局拔高到了意识形态和政治取向上。</p><p><strong>汪小菲实现“微博自由”？</strong></p><p>网民将这桩离婚案高度“政治化”确也不是捕风捉影。今年6月5日大S方面爆出准备离婚当天，网民扒出了同天汪小菲在微博上痛斥台湾政府的言论，其中一条关于确诊台湾人跑去厦门的微博上，他更怒不可遏的夹带粗话形容对方是“汉奸”。</p><div id="innity-in-post"></div><div id="dfp-ad-midarticlespecial">        </div><p>根据微博的编辑记录，汪小菲对这则微博的措辞前后改了三次，最终才将“这帮汉奸”改为“这一笔”，“汪小菲逐渐文明”的话题也一度冲上热搜。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/2021-11/weiboziyou.jpg?VersionId=IYCBju8JEdks9ZvRGBnpkw4z1V2Vpi2J" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>汪小菲微博的措辞前后修改三次，“汪小菲逐渐文明”的话题也一度冲上热搜。（互联网）</figcaption></figure>                  <p>有趣的是，汪小菲隔日再发微博解释，“因为实在担心家人，昨天说了一些不好的话”“疫情期间确实情绪容易激动”；当晚又发微博称“爱所有的家人，我老婆是第一”，被舆论解读为隔空向大S道歉，挽救婚姻，还被网民调侃为“满满的求生欲”。</p><p>汪小菲的微博一度被指“当成朋友圈在用”，意思是他常常在这样一个人人可见的公开平台上发表爱国立场鲜明且涉及敏感话题的观点。他自己似乎也曾意识到这点，因此在2015年2月发文称，以后再也不发表对时事的看法了。</p><p>但2019年汪小菲又按捺不住“重出江湖”，感叹两相对比，还是大陆的大环境好。去年年初，他还上传了一段视频称因自己讲北京话而被台湾出租车司机爆粗，并称台湾如今是一个“充满仇恨的地方”，随后又删掉了视频。</p><p>汪小菲的这些“纠结”都被网民看在眼里。昨天离婚消息一出，许多网民都为汪小菲“欢呼”，恭喜他终于实现“微博自由”，还有网民说，“汪小菲连爱国自由都没有，这个婚姻也没有持续的意义了”。更有网民操心地喊道：“孩子必须弄回来，当个堂堂正正的中国人，不能让他们在台湾受教育！”</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/2021-11/das_wangxiaofei.jpg?VersionId=sQKXhADGzTHomPy0Sou6UWx4WZ2b.t_r" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>大S和汪小菲育有一女一子。（互联网）</figcaption></figure>                  <p>作为曾经的“京城四少”，“根正苗红”的汪小菲或许认为他发表的两岸言论并无不妥之处，但作为一个台湾女婿，隔三差五的影射民进党“道貌岸然”，又疾呼“领土不容分割”，还转发“一点儿也不能少”的微博，对深耕台湾演艺圈的妻子大S来说，无疑是一种巨大的压力。</p><p><strong>两岸婚姻生活被放大解读</strong></p><p>平心而论，明星婚姻并非完全不沾人间烟火，一般夫妻所要考虑的“财米油盐”他们也会遇到。</p><p>一直北京、台北两头跑的汪小菲曾在2015年12月的台湾综艺节目《康熙来了》里透露，夫妻俩会就“在哪里生活、哪里带孩子（上学）、小孩教育方面”有分歧。这其实是任何婚姻、家庭中太正常不过的事，就算夫妻俩都生在中国大陆，“地域之争”也不少见。而疫情使跨境旅行困难重重，以前还可以克服的地域问题如今变得更加棘手，自然也会加剧冲突。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/2021-11/screenshot_2.jpg?VersionId=kcOjIJqUiRsrCELzCjZg6uzVHdhVhNnP" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>汪小菲曾在2015年12月播出的《康熙来了》里透露，夫妻俩会就“在哪里生活、哪里带孩子（上学）、小孩教育方面”有分歧。（互联网）</figcaption></figure>                  <p>但问题在于，随着大陆“武统”声量的高涨，台湾“抗统”情绪的浓烈，两岸民意的对立越发尖锐，两岸联姻的明星夫妻，其一言一行也就越被舆论刻意放大往意识形态方向解读。</p><p>“大S不带孩子去大陆，选择在台湾上学”这点就曾被大陆网民质疑是“鄙视大陆”。台湾今年年中急缺疫苗期间，大S曾在INS上发泄不满，但最终还是没有前往大陆打针，也被网民拿出来指摘。至于汪小菲批评台独的言论，则让他被一些台湾网民视为“全民公敌”。</p><p><strong>10年婚姻走过两岸关系的高低潮</strong></p><p>10年前那场在海南三亚举办、轰动一时的大汪婚礼还历历在目。彼时，正寻求连任的台湾总统马英九刚签署了两岸经济合作框架协议（ECFA），在他执政的八年中，两岸通过海基会、海协会达成了多项协定。两岸关系就如新婚燕尔的大S和汪小菲一般处于甜蜜期。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/2021-11/jiehun.jpg?VersionId=Ov0kKzzjBRFQQrNTCyLHG8WbeFde3JgD" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>2011年大S和汪小菲在海南三亚举办婚礼，轰动一时。（互联网）</figcaption></figure>                  <p>10年后，大汪离婚，蔡英文连任，两岸关系不复从前，再伴随着大国博弈滋生出的民族情绪，两岸民意高度对立，把官方都还未想要完全抹掉的“模糊空间”逼到了死角，涉及两岸的公众人物一不小心就会被贴上“独”或“统”的标签，这也是为何大汪离婚会被大陆“小粉红”们包装成独统之间的对决。一对明星夫妻的散场成了小粉红“出征”的战场，让人不胜唏嘘。</p><p>上周举行的中美元首视频会晤上，中国国家主席习近平在阐述台湾问题立场时，首次提到北京在这一问题上有“耐心”，“愿以最大诚意、尽最大努力争取和平统一的前景”，似是回应当前大陆内部高涨的武统声音。</p><p>在大S的妹妹、同样是台湾艺人的小S因称台湾奥运选手为“国手”而被指为“台独”时，大陆国台办和大陆官媒也多次发声呼吁对台独要“精准辨识、打击”“擦亮眼睛”“不要制造对立，上了真台独的当”。</p><p>但显然这些声音不容易被“小粉红”们听见，他们享受的可能只是借着明星的聚光灯，扮成冲锋在统战一线的那份快感。</p>
