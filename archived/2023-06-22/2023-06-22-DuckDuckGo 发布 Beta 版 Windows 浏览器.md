[DuckDuckGo 发布 Beta 版 Windows 浏览器](https://www.solidot.org/story?sid=75318)
------

<div>
DuckDuckGo 发布了 Beta 版 Windows 浏览器。DuckDuckGo 是在去年 4 月发布了一款注重隐私的 Mac 版浏览器，不同于其它浏览器主要是基于 Google Chromium，DuckDuckGo 的 Mac 版浏览器是基于苹果的 WebKit 渲染引擎。它的 Windows 版本也是利用了操作系统自带的引擎，通过 Windows WebView2 调用使用 Blink 渲染引擎。在大部分网站它会被视为是 Microsoft Edge。DuckDuckGo 浏览器内置的播放器会过滤掉 YouTube 上隐私侵犯的广告，会预屏蔽跟踪程序，数据使用比 Chrome 少六成。目前浏览器的一大缺陷是不支持扩展，DuckDuckGo 表示计划未来加入扩展支持。<p></p>
</div>
