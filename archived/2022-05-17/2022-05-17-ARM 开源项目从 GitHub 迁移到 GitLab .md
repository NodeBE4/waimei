[ARM 开源项目从 GitHub 迁移到 GitLab](https://www.solidot.org/story?sid=71554)
------

<div>
ARM 开源项目从 GitHub 迁移到 GitLab。GitLab 官方博客阐述了选择 GitLab <a href="https://about.gitlab.com/blog/2022/05/17/arm-open-source-makes-a-seamless-migration-to-gitlab/" target="_blank">作为自托管平台的优点</a>：最大化选择和成本效益，最小化供应商锁定。使用 GitLab 将允许 ARM 使用自己的工具，不像 GitHub 是一个黑箱。ARM 芯片占到了移动市场的 95%，而 ARM 公司的业务是 IP 授权，大部分核心基础设施部署在私有硬件的，安全对其至关重要。
</div>
