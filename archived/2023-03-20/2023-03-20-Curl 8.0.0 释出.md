[Curl 8.0.0 释出](https://www.solidot.org/story?sid=74441)
------

<div>
庆祝项目诞生 25 周年，curl 作者 Daniel Stenberg 宣布释出 v8.0.0 ，称该版本没有 API 或 ABI 破坏，也就是不会造成向后兼容性问题。该版本修复了 6 个新漏洞，其中 5 个危险等级低，另外一个为中等。它的另一个主要变化是构建系统不再支持缺乏可工作 64 位数据类型的系统，其它主要是 bug 修复。<p></p>
</div>
