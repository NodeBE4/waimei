[近五分之一 Microsoft SQL Server 已终止支持](https://www.solidot.org/story?sid=78456)
------

<div>
互联网上的大量 Microsoft SQL Server 服务器早已达到寿命终点。IT 资产管理平台 Lansweeper 扫描了逾百万  SQL Server 实例，发现其中 19.8% 的版本微软已经终止支持。还有 12% 的实例运行 SQL Server 2014，该版本将于今年 7 月 9 日终止扩展支持，意味着届时停止支持的 SQL Server 版本比例将达到 32%。虽然客户可以选择付费获得 SQL Server 2014 的安全更新三年，但这一发现凸显企业依赖过时软件以及升级到支持版本的难度。<p></p>
</div>
