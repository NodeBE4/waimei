[微软有了自己的 Linux 发行版](https://www.solidot.org/story?sid=68310)
------

<div>
微软有了自己的 Linux 发行版 <a href="https://github.com/microsoft/CBL-Mariner" target="_blank" rel="noopener noreferrer nofollow" data-component="externalLink">Common Base Linux, (CBL)-Mariner</a>，软件巨人悄无声息的在 GitHub 上公开了 <a href="https://github.com/microsoft/CBL-Mariner">CBL-Mariner 1.0</a> 版本的源代码。CBL-Mariner 不是桌面 Linux 而是服务器端 Linux，它在微软内部被用于云基础设施以及边缘计算产品和服务。它<a href="https://www.zdnet.com/article/say-hi-to-microsofts-own-linux-cbl-mariner/" target="_blank">提供了</a>一组云和边缘服务所需的通用软件包集，可以很容易根据需要引入额外的软件包。它是一个轻量级的发行版，可作为容器或容器主机使用。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
