[亚马逊开源 Lumberyard，由 Linux 基金会托管](https://www.solidot.org/story?sid=68221)
------

<div>
亚马逊<a href="https://aws.amazon.com/cn/blogs/gametech/open-3d-engine/">宣布</a>开源 Lumberyard，该项目将由 Linux 基金会托管，它是首个开源的功能完整的 3A 级游戏引擎。Lumberyard 是基于 Crytek 的引擎 CryENGINE，此前也允许开发者访问源代码，但有限制并不公开。亚马逊开源的是 Lumberyard 的一个新版本，将其更名为 <a href="https://o3de.org/" target="_blank">Open 3D Engine</a>，许可证切换到了 Apache 2.0 license 或 MIT，允许不受限制的任意使用，<a href="https://github.com/o3de/o3de" target="_blank">源代码</a>发布在 GitHub 上。Linux 基金会<a href="https://www.linuxfoundation.org/press-release/linux-foundation-to-form-new-open-3d-foundation/">宣布</a>成立 Open 3D 基金会，其创始成员包括了 Adobe、AWS、华为、Niantic、英特尔和 Red Hat 等。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
