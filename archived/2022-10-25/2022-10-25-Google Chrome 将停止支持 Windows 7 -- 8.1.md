[Google Chrome 将停止支持 Windows 7 / 8.1](https://www.solidot.org/story?sid=73162)
------

<div>
Google <a href="https://www.bleepingcomputer.com/news/google/google-chrome-to-drop-support-for-windows-7-81-in-feb-2023/">宣布</a>其 Chrome 浏览器将从 2023 年 2 月起停止支持 Windows 7 / 8.1。Google Chrome 支持经理称，计划于明年 2 月 7 日发布的 Chrome 110 将终止对 Windows 7 / 8.1 的支持，用户设备需要 Windows 10 及以上版本才能运行  Chrome 未来版本。Google 终止支持的决定与微软  Windows 生命周期政策一致，软件巨人计划于 2023 年 1 月结束 Windows 7 / 8.1 的扩展安全更新支持。目前 Windows 7 在操作系统市场的占有率仍然逾 10%，而 Windows 8.1 只有 2.7%。
</div>
