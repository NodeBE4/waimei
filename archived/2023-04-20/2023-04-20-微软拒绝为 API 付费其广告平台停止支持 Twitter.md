[微软拒绝为 API 付费其广告平台停止支持 Twitter](https://www.solidot.org/story?sid=74721)
------

<div>
微软拒绝为访问 API 支付高昂的费用，它通知客户其广告平台将从 4 月 25 日起停止支持 Twitter。Twitter CEO 马斯克（Elon Musk）随后威胁要起诉微软。Twitter 废除了免费 API 访问，从 4 月 29 日起启用新的付费 API，其中企业级方案起步价高达每月 4.2 万美元。Twitter 的这一决定已经促使很多独立开发者和大型企业关闭基于 Twitter 的应用或放弃整合 Twitter。美国国家气象局的紧急警报系统也将在关闭 API 访问之后停止支持 Twitter。<p></p>
</div>
