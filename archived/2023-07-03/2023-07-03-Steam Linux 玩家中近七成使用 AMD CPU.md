[Steam Linux 玩家中近七成使用 AMD CPU](https://www.solidot.org/story?sid=75402)
------

<div>
根据 Steam 的 2023 年 6 月软硬件调查，Linux 玩家规模依旧非常小，只占 1.44%，其中四成使用的是 Valve 的掌机 Steam Deck。Steam Deck 是基于 AMD 的 APU ，在 Steam 上的所有 Linux 玩家中有  67% 使用了 AMD CPU，英特尔 33%。相比之下，去年 6 月 Linux 玩家中 AMD CPU 的比例只有 45%，英特尔是 54%。而在 Windows 玩家中英特尔的市场份额为 68%，AMD 只有 32%。Steam Deck 是 Linux 玩家大量使用 AMD CPU 的主要推动力量。AMD 的 GPU 对 Linux 的支持也相当出色，它的显示驱动开源，包含在最新版的 Linux 内核，Linux 用户基本上可以开箱即用 AMD GPU，无需额外设置。<p></p>
</div>
