[Chrome 113 将支持 WebGPU](https://www.solidot.org/story?sid=74599)
------

<div>
Google 宣布 Chrome 的下一个版本 v113 将支持 WebGPU API。WebGPU 为 Web 提供了下一代高性能图形 API，允许在 GPU 上执行渲染和计算操作，类似 Direct3D 12、Metal 和 Vulkan。与现有的源自 OpenGL (ES) 的 WebGL API 不同的是，WebGPU 提供了对更高级的 GPU 功能的访问。Chrome 113 的 WebGPU 支持将只限于 ChromeOS、macOS 和 Windows 三个平台，Linux 发行版的支持要等待今年晚些时候。<p></p>
</div>
