[树莓派推出 Raspberry Pi 500 和 Raspberry Pi Monitor](https://www.solidot.org/story?sid=80001)
------

<div>
树莓派宣布推出 Raspberry Pi 500 和 Raspberry Pi Monitor，用户只需花 190 美元就能组装出一台简单的 ARM 桌面 PC。Raspberry Pi 500 是四年前推出的 Raspberry Pi 400 的后续型号，价格从 70 美元涨到 90 美元，它是一台键盘形状的一体机，可以外接 Raspberry Pi Monitor 作为显示器。其 CPU 为四核 2.4GHz Arm Cortex-A76 处理器，8GB LPDDR4X-4267 内存，VideoCore VII GPU，双 4Kp60 HDMI 显示输出，双频 802.11ac Wi-Fi 和蓝牙 5.0， 2 个 USB 3.0 端口，1 个 USB 2.0 端口，千兆以太网端口，水平 40 针 Raspberry Pi GPIO 连接器。Raspberry Pi Monitor 售价 100 美元，为 15.6 英寸 1080p IPS 显示器。用户可选配 120 美元桌面配件，包括鼠标、USB-C 电源、micro HDMI 转 HDMI 适配器和初学者指南。<p></p>
</div>
