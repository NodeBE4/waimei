[Ubuntu 是 Canonical 公司的产品](https://www.solidot.org/story?sid=74271)
------

<div>
Canonical 公司在 Ubuntu 中加入了愈来愈多恼人的广告，这也从侧面证明了 Ubuntu 不只是 Canonical 支持的 Linux 发行版，而是该公司的产品。既然是产品，那么其存在的最终目的就是给 Canonical 公司带来收入。而加入广告意味着目前 Ubuntu 还没能让 Canonical 赚到足够的钱。对于那些不太可能增加收入的用户的观点，Canonical 是不感兴趣的。对普通用户而言，Ubuntu 是基于 Debian，还有很多社区发行版是基于 Ubuntu，因此切换到其它 Linux 版本的难度并不大。<p></p>
</div>
