[Firefox 101 释出](https://www.solidot.org/story?sid=71693)
------

<div>
在 Chrome 102 发布一周之后，Mozilla 释出了 <a href="https://www.mozilla.org/en-US/firefox/101.0/releasenotes/" target="_blank">Firefox 101</a>(直接<a href="https://ftp.mozilla.org/pub/firefox/releases/101.0/">下载</a>)。按照这一速度，Firefox 在版本号上将很快超过 Chrome。Google 在三月底发布 Chrome 100，Mozilla 在五月初<a href="https://www.solidot.org/story?sid=71421">释出</a> Firefox 100。Firefox 101 在开发者方面的<a href="https://developer.mozilla.org/en-US/docs/Mozilla/Firefox/Releases/101">变化</a>包括：默认 CSS prefers-contrast 媒体功能，三种新的 CSS 视窗大小，支持 HTMLInputElement.showPicker()，支持可构造样式表，正确解析 AV1 编解码器参数，等等。
</div>
