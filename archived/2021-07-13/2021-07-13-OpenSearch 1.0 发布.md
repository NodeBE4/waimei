[OpenSearch 1.0 发布](https://www.solidot.org/story?sid=68269)
------

<div>
亚马逊 AWS 发起的 Elasticsearch 分支 OpenSearch <a href="https://opensearch.org/blog/updates/2021/07/opensearch-general-availability-announcement/">释出了 1.0 版本</a>。今年初，开发 Elasticsearch 和 Kibana 的公司 Elastic <a href="https://www.solidot.org/story?sid=67473" target="_blank">宣布</a>更改许可证，新版本将不再使用 Apache 2.0 而是使用 Elastic License 和 Server Side Public License，此举旨在禁止云服务商如 AWS 使用它的软件作为一种服务提供给客户。但许可的更改也意味着 Elasticsearch 和 Kibana 不再是开源软件了。OpenSearch 1.0 是基于  Apache 2.0 授权的 Elasticsearch 7.10.2 &amp; Kibana 7.10.2，包含了搜索引擎守护程序 OpenSearch，可视化和用户界面 OpenSearch Dashboards，以及安全、警告和异常监测等高级功能。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
