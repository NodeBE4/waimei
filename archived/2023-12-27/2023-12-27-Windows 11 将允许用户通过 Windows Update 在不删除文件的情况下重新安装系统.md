[Windows 11 将允许用户通过 Windows Update 在不删除文件的情况下重新安装系统](https://www.solidot.org/story?sid=76987)
------

<div>
重新安装 Windows 11 操作系统从来不是简单的事，微软也注意到了，它在最新的 Windows Insider beta 版本中加入了“使用 Windows Update 修复问题”的功能。该功能仍然在开发之中，还无法使用。根据微软的描述，该功能将在保留应用、文件和设置的情况下重新安装当前操作系统。差不多就是一种不删除文件全新安装操作系统的方法。重新安装系统通常有助于解决很多系统故障。由于它还在开发中，该功能的很多信息尚无法确定。<p></p>
</div>
