[微软宣布运行在云端的 Windows 365](https://www.solidot.org/story?sid=68289)
------

<div>
微软<a href="https://www.microsoft.com/en-us/microsoft-365/blog/2021/07/14/introducing-a-new-era-of-hybrid-personal-computing-the-windows-365-cloud-pc/">宣布</a>了运行在云端的<a href="https://news.microsoft.com/2021/07/14/microsoft-unveils-windows-365-ushering-in-a-new-category-of-computing/" target="_blank"> Windows 365</a> 操作系统（或远程桌面），允许用户从任何设备上通过云端选择使用  Windows 10 或 Windows 11。该服务将于 8 月 2 日正式推出。运行在微软云服务的操作系统将能安全串流完整的 Windows 体验，包括应用、数据和设置。用户可通过 Mac、iPad、Linux 和 Android 设备访问 Windows 365。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
