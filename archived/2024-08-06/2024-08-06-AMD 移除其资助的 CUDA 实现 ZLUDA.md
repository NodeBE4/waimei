[AMD 移除其资助的 CUDA 实现 ZLUDA](https://www.solidot.org/story?sid=78902)
------

<div>
今年二月，ZLUDA 项目开发者  Andrzej Janik 在 GitHub 上开源了 AMD 资助的 CUDA 实现，允许 CUDA 应用无需修改就能运行在 AMD 的显卡上。AMD 停止了对改项目的资助，但允许开发者公开源代码。但开源六个月之后，AMD 律师撤销了开源的决定，项目也随之从 GitHub 下架。Andrzej Janik 证实，该项目没有收到英伟达的任何法律威胁，而是应 AMD 法务部门的要求。他表示自己还完成了一个运行在 AMD GPU 上的 NVIDIA GameWorks 实现，但相关代码永远不会开源了。<p></p>
</div>
