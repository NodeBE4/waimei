[Mozilla 考虑延长支持 Windows 7/8/8.1](https://www.solidot.org/story?sid=78648)
------

<div>
微软已经停止支持 Windows 7/8/8.1 操作系统，而操作系统上最流行的应用——即浏览器如 Google Chrome 和 Microsoft Edge 也都在去年初停止了对这些旧操作系统的支持，Mozilla 去年 7 月释出的 Firefox 115 ESR 是支持 Windows 7/8/8.1 的最后一个版本。在 Reddit 上进行 AMA 活动时，Mozilla 开发者透露他们正在考虑延长对 Windows 7/8/8.1 的支持，但目前尚未决定支持多长时间。Mozilla Firefox 的市场份额在逐渐缩小，此举可能旨在保留住现有的用户基数。<p></p>
</div>
