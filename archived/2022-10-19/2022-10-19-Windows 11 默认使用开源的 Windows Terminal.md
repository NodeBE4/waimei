[Windows 11 默认使用开源的 Windows Terminal](https://www.solidot.org/story?sid=73103)
------

<div>
Windows Terminal 项目经理 Kayla Cinnamon <a href="https://devblogs.microsoft.com/commandline/windows-terminal-is-now-the-default-in-windows-11/" target="_blank">宣布</a>，从  Windows 11 22H2 开始所有命令行应用将默认用 Windows Terminal 打开。Windows Terminal 是微软在 2019 年宣布的新命令行终端，<a href="http://www.github.com/microsoft/terminal">源代码</a>发布在 GitHub 上。Windows Terminal 支持多标签、Unicode 和 UTF-8 字符、GPU 加速 DirectWrite/DirectX 文本渲染引擎，定制主题、样式和配置。
</div>
