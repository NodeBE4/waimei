[AMD 和 Valve 合作改进 Linux 下的 CPU 性能调整](https://www.solidot.org/story?sid=68445)
------

<div>
Valve 准备在今年年底发售配备 AMD APU 的 Linux 掌机，利用 Steam Play (Proton / Wine) 兼容层运行原本只支持 Windows 操作系统的游戏。但掌机必须为延长使用时间而进行性能方面的优化，但 AMD 调整 CPU 频率和功耗的驱动  ACPI CPUFreq 在 Linux 下工作并不理想。现在 AMD 和 Valve <a href="https://www.phoronix.com/scan.php?page=news_item&amp;px=AMD-Valve-New-CPU-Freq">正在合作改进该方面的工作</a>。AMD 将在下个月的  XDC 大会上公布更多细节。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
