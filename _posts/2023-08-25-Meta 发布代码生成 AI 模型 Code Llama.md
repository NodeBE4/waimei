[Meta 发布代码生成 AI 模型 Code Llama](https://www.solidot.org/story?sid=75891)
------

<div>
Meta 发布了它的代码生成 AI 模型 Code Llama。类似 GitHub Copilot 和 Amazon CodeWhisperer，以及 StarCoder、StableCode 和 PolyCoder 等开源 AI 代码生成模型，Code Llama 支持为 Pytho、C++、Java、 PHP、Typescript (Javascript)、C# 和 Bash 等编程语言补完代码和调试。Code Llama 是 Llama 2 的编程专用版本，有 7B、13B 和 34B 参数三个版本，使用 500B tokens 的代码和代码相关数据进行训练。其中 7B 参数的版本可以单张 GPU 上运行，7B 和 13B 参数版本更适合实时代码补完之类的任务，而 34B 参数版本更适合作为编程助手使用。Code Llama 采用和 Llama 2 相同的许可证，可作为研究和商业使用。<p></p>
</div>
