[AWK 作者为语言加入 Unicode 支持](https://www.solidot.org/story?sid=72576)
------

<div>
AWK 作者、普林斯顿大学教授 Brian Kernighan 在接受<a href="https://youtu.be/GNyQxXw_oMQ?t=462">采访</a>（YouTube，8 分钟左右）时透露他最近做了点一个小小的工作，为 AMK <a href="https://www.theregister.com/2022/08/23/universal_unix_tool_awk_gets/">加入</a>了 <a href="https://github.com/onetrueawk/awk/commit/9ebe940cf3c652b0e373634d2aa4a00b8395b636" target="_blank">Unicode 支持</a>。AWK 语言被用于分析文本文件，是 Unix 操作系统的核心组件之一，任何 POSIX 兼容操作系统被认为都应该包含 AWK。它的名字来自于三名原作者 Alfred <b>A</b>ho、Peter <b>W</b>einberger 和 Brian <b>K</b>ernighan 姓的首字母缩写。Kernighan 教授同时也是著名《C 程序设计语言》一书作者 K&amp;R 之一的 K，另一个 R 就是 C 作者 Dennis Ritchie。他命名了 Unix 系统，并发明了 "Hello, world" 的语言演示。原版的 AWK 在 GitHub 上被称为  <a href="https://github.com/onetrueawk/awk" rel="nofollow">One True AWK</a>。今年 5 月 80 岁的 Kernighan 教授在邮件中表示他在学习 Git 系统，之后会递交 pull request 为 AMK 加入 Unicode 支持。
</div>
