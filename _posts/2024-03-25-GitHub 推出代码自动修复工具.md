[GitHub 推出代码自动修复工具](https://www.solidot.org/story?sid=77684)
------

<div>
GitHub 推出了代码自动修复工具 Code Scanning Autofix。该功能目前处于公测阶段，提供给 GitHub Advanced Security 客户，在其私有库上自动启用，能帮助处理 JavaScript、Typescript、Java 和 Python 中逾九成的警告类型。GitHub 称该功能在不怎么需要开发者编程的情况下修复逾三分之二发现的漏洞。GitHub 称，对于支持的语言中发现的漏洞，修复建议将包括建议修复的自然语言解释，以及开发者可接受、编辑或忽略的代码建议预览。软件公司 Functional Software 也推出了自己的 AI 代码自动修复工具。<p></p>
</div>
