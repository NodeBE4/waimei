[微软给 Windows 11 设置面板加入了广告](https://www.solidot.org/story?sid=74911)
------

<div>
微软正在给 Windows 11 加入愈来愈多的广告。在新的测试版本中，设置面板页面出现了微软产品的广告。根据用户在社交网络上公布的 Canary 渠道早期测试版本的截图，面板的顶部出现了 Microsoft 365 的广告，告诉用户可以免费试用。下面还有一个“完成账户设置”的提示，这里的账户指的是 Microsoft Account。其它公布的截图也与 Microsoft Account 有关。<p></p>
</div>
