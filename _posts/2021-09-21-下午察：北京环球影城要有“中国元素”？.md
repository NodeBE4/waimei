<p>中美关系冰冻，中国民众对美国影视文化却热情不减，昨天开园的北京环球影城门票一分钟售罄。众人在阴雨连绵的中秋佳节前夕蜂拥而至，一些人还穿着哈利波特服饰。</p><p>有中国人愿意排队90分钟体验哈利波特的禁忌之旅，也有中国人对北京环球影城不屑一顾。</p><p>一名微博用户就发飙：“国内五千年的文化底蕴，做不出来一个火爆的主题公园？”还有其他网民应和：“一个劲的吹捧国外的！”“对美国文化不感兴趣！”“这是一种文化侵蚀！”</p><section id="imu"><div id="dfp-ad-imu1">        </div></section>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-09/133711210.jpg?h5NYLBtQ1yXBKMjZqLLzDZj_PbbXTsbG&itok=GwSY3AoT" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>一些中国网民对北京环球影城不屑一顾，但也有游客在度假区留影。（中新社）</figcaption></figure>                  <p>中国官媒《环球时报》总编胡锡进昨夜在微博评论说，香港、上海迪士尼乐园与北京环球影城三个大型“洋主题公园”在中国落户，折射了中国经济进步和国民消费能力不断上升，“随着中国不断发展，国民视野同步开阔，民族文化自信也在上升，西方文化符号在中国社会中的吸引力应当说也过了前些年的巅峰期。然而，一些西方文化符号在中国仍有很大粉丝群，整个中国社会也对它们持友好态度。”</p><p>在胡锡进看来，“洋主题公园”能持续在中国凝聚人气，“反映的是中国社会对外部文化的开放、友好态度，这就是中国文化的整体包容性”。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-09/file7hmbnclfjqselgvijt2_1.jpg?sLZz3hErRLgiRsF5VCRuFksvmhEWok4L&itok=cydVd6B-" pinger-seen="true" <p>不过，留言区有不少反对他甚至谩骂的声音，有网民批胡锡进立场摇摆、人格分裂，有网民质问：“那里面不都是你老胡每天痛骂的美帝糟粕文化吗？”还有的调侃：“环球影城是《环球时报》办的？”</p><p>环球影城离不开受美国资本主义灌溉的好莱坞影视娱乐，属美国大众文化的板块。而美资全球扩张所带动的文化输出，让环球影城享誉国际，深入人心。中国民众对此的热衷程度，实际体现在访客量，也似乎不受中美关系恶化、国潮风高涨影响。至于这种热度是展示了“中国文化包容性”，还是纯粹被美国大片里的角色与故事吸引，则见仁见智。</p><div id="innity-in-post"></div><div id="dfp-ad-midarticlespecial">        </div><p><strong>北京环球影城的前世今生</strong></p><p>首座环球影城——好莱坞环球影城——早在1964年就开园了，起初是为了让游客参观摄影棚和场景而设立的。多年后逐渐演变成如今的主题乐园，包括娱乐设施、酒店等。</p><p>许多到访美国的游客慕名而至，公司的标语是“洛杉矶娱乐之都”。随着好莱坞电影国际化，环球影城的招牌也享誉世界。位于日本大阪市与新加坡圣淘沙的环球影城，分别于2001年与2011年开业。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-09/wizarding-world-harry-potter-universal-studios-hollywood.jpg?l7T5I2pVFCLlotaOevPKxKYCJBYd8Qw6&itok=xoBzOPkQ" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>1964年开放的好莱坞环球影城是全球第一座环球影城。（互联网）</figcaption></figure>                  <p>据《新京报》的梳理，早在2000年左右，北京就认为该市要发展娱乐旅游，从国外引进大型主题公园是最有效的。</p><p>在中国加入世界贸易组织（WTO）的2001年，关于北京环球影城的磋商紧锣密鼓。环球影城主题公园高层代表团当年10月与北京市签署《合资协议》，期间也做了大量的市场调研与规划。2002年底，主题公园的可行性研究完成，并于次年将项目建议书上报国家发展计划委员会，但“由于国家宏观调控政策，项目审批进程暂缓”。这一缓，就是五年。</p><p>直到2008年，北京市政府决定将环球影城项目作为拉动“后奥运北京”经济发展及旅游业结构调整升级的重点项目，再次上报国务院，但项目又多次搁浅。又过五年，2013年6月，国务院才同意北京市建设环球影城项目的意向。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-09/file7henoe6fi5t8q9qxac3.jpg?eU0u2VQdHNjm4S0jBwQ2umZuBxOYZjMb&itok=JGPRpJDL" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>北京环球影城到了2018年7月27日才正式开工建设。（中新社）</figcaption></figure>                  <p>又等了五年，北京环球影城终于在2018年7月27日正式开工建设。为何项目如此命运多舛？官方没给明确答案，但据《中国房地产报》报道，2009年时任北京市发改委主任的张工说，国务院方面希望进一步压缩项目投资规模，并加入更多中国元素。另据《北京日报》报道，中共北京市委书记蔡奇上周四（16日）会见环球影城母公司、美国康卡斯特集团董事长兼首席执行官罗伯兹时，也提出希望园区“融入更多中国元素”。</p><p><strong>有哪些中国元素？</strong></p><p>据媒体上述报道，中国官方在北京环球影城开建前与建成后，已两次提出要融入中国元素。如今园区里有什么中国元素呢？</p><p>官方微博介绍，园区设施景点“反映中国悠久的文化传承”。官方网站介绍，园内有七个主题景区：哈利波特的魔法世界、变形金刚基地、功夫熊猫盖世之地、好莱坞、未来水世界、小黄人乐园、侏罗纪世界努布拉岛。</p><p>虽然“熊猫”与“功夫”确实属中国元素，但该景区的母体——美国动画电影系列《功夫熊猫》，依旧是好莱坞对中国文化的诠释，希望借此打入中国市场，并将美国观众所熟知的“熊猫”与“功夫”融合，两边通吃。电影当年在中国首映时，还遭到一些人的批评与抵制。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-09/d494-9fa4ac8d6fbeedb166ebb6f0b4d954b5.png?rbEkkpVG5KQqxgeBEDer0dUjSIrQ1fTH&itok=oOE1u08b" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>北京环球影城中的功夫熊猫景区。（互联网）</figcaption></figure>                  <p>别的中国元素，就是附属于其他景区的细节，如小黄人乐园售卖的十二生肖系列产品、变形金刚景区的中国文化展示区等。其实，这些元素，是任何跨国企业在地化策略所涵盖的常规操作，如麦当劳在各国都会融入一些当地食物元素，但汉堡薯条的快餐文化仍是主旋律。而且，当地文化之只能点缀，不能“喧宾夺主”，不然就会影响原有的吸引力。</p><p>中国未来研究会旅游分会副会长刘思敏说，环球影城在代表美国娱乐文化最高水平的同时，更是伴随着全球化而产生的重要影响，其契合了现代人的精神需求，“中国元素的加入也要有一定的把握和衡量。如果做成了北京的‘唐人街’，中国老百姓又怎么愿意去那里旅游度假呢？”</p><p>刘思敏说：“至于这样的中国元素能不能被受众市场所接受，就等看开园后由旅客们来投票了。”——投票结果现在还没看到，但根据网民的讨论与中国媒体的报道，目前最受欢迎的景区应是哈利波特的魔法世界。</p> src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-09/file7hmtiozg7yvj39ri53g.jpg?4zA3p1kGFj0DR9I5SSgpemwKdVuAgtjd&itok=Ug6w0Sd0" pinger-seen="true" <p><strong>未来真正的中国元素主题公园？</strong></p><p>万达集团董事长王健林曾说：“作为中国首都和文化中心，在北京建环球影城主题公园项目欠妥。上海已经做了迪士尼主题公园项目，北京如果再建环球影城项目，意味着中国南北两大城市文化项目全部被美国元素所主导，搞得再好又有什么意义？因为不是中国的东西。”</p><p>“迪士尼实在不该来大陆”，王建林2016年5月22日在央视节目抛出豪言壮语：“我们万达会让迪士尼中国在未来20年都无法盈利。”</p><p>事实证明，上海迪士尼于2016年6月16日正式开业，仅一年后就实现盈利，这在其他国家的迪士尼也实属罕见。而王建林投入重资、要与迪士尼一争高下的的万达乐园如今销声匿迹。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-09/file6udngqc1xbth4zxchxk.jpg?3K9tF_w6asCdM29g6aKTyLS8ofwVQjQ0&itok=p1GzIpD1" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>王健林曾扬言，万达会让迪士尼中国在未来20年都无法盈利。（中新社）</figcaption></figure>                  <p>正如搜狐用户“我爱学习俱乐部”分析，迪士尼凭借的超级IP吸引观众，而媒体网络、主题公园及度假村、影视娱乐、消费品以及互动娱乐五大业务板块，为这些IP构建了可以流转、增值的生态版图。而环球影城也是类似。</p><p>分析认为，中国主题公园管理层最缺乏的是对主题公园核心要素的了解，更缺乏对主题公园文化内涵的理解，尤其是对IP不够重视。</p><p>因此，原创优质的内容配上完整高能的产业链，是主题公园的成功基础，当然也少不了政策与资本的扶持，以及多个行业共同发展、互相协作产生共振。中国文化底蕴雄厚，一些国漫制作已有出色的成绩，但真正以中国元素主导的世界级主题公园的发展，恐怕还需要时间。</p><p>而在当下，一些中国未成年人在玩网游、刷抖音的时间被限制、崇拜的明星不再露脸后，或许会闹着让父母带他们去北京环球影城逛逛。</p>
