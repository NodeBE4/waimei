[Linux 桌面操作系统市场份额突破 4%](https://www.solidot.org/story?sid=77512)
------

<div>
根据 StatCounter 的最新统计数据，Linux 桌面操作系统市场份额突破 4% 达到 4.03%。它对 Linux 桌面系统的统计没有包含 Chrome OS，而 Chrome OS 是基于 Linux 的操作系统，早期是基于 Ubuntu、Gentoo 和 Debian 发行版。如果将 Chrome OS 包含在内，Linux 的份额增加到 6.3%，次于 Windows 的 72.13% 和 OS X 的 15.46%。不包含 Chrome OS 的 Linux 桌面份额是在 2023 年 6 月首次突破 3%，仅仅 8 个月增加 1% 这一数字还是令人惊讶的。<p></p>
</div>
