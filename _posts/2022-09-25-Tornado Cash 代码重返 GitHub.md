[Tornado Cash 代码重返 GitHub](https://www.solidot.org/story?sid=72870)
------

<div>
上个月美国财政部宣布制裁被控为朝鲜黑客组织 Lazarus Group 洗钱的加密货币混币器 Tornado Cash，禁止美国个人或组织与 Tornado Cash 有任何交易往来。中心化源码托管平台 GitHub 随即下架了 Tornado Cash 代码库，删除了开发者账号。美国财政部上周<a href="https://www.solidot.org/story?sid=72769">澄清</a>，对 Tornado Cash 的制裁不涉及源代码。本周在开发者的呼吁之后 GitHub <a href="https://slashdot.org/story/22/09/24/1741217/crypto-mixing-service-tornado-cash-code-is-back-on-github">恢复</a>了 <a href="https://github.com/TornadoCash">Tornado Cash</a> 项目，多位项目开发者的账号也都恢复。
</div>
