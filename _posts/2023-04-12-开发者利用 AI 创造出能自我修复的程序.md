[开发者利用 AI 创造出能自我修复的程序](https://www.solidot.org/story?sid=74643)
------

<div>
调试程序找出 bug 的过程是令人沮丧的，为什么不让 AI 来帮助你完成这个工作？一位用户名为 BioBootloade 的开发者创造出能帮助 Python 程序自我修复的程序  Wolverine。代码发布在 GitHub 上。使用 Wolverine下需要用户拥有 OpenAI 的 GPT-3.5 或 GPT-4 的 API key。GPT 3.5 API 目前开放给所有用户使用，但 GPT-4 的访问是受限的。<p></p>
