[WordPerfect 移植到 Linux](https://www.solidot.org/story?sid=72204)
------

<div>
在将 Lotus 1-2-3<a href="https://github.com/taviso/123elf" target="_blank"> 移植</a>到 Linux 之后， Google Project Zero 团队的白帽子黑客 Tavis Ormandy 又将 1990 年代的经典字处理软件 WordPerfect <a href="https://www.theregister.com/2022/07/20/wordperfect_for_unix_for_linux/?td=rt-3a" target="_blank">移植到 Linux</a>。<a href="https://github.com/taviso/wpunix">源代码</a>发布在 GitHub 上，移植的版本是  WordPerfect version 7 for Unix。他还将其打包到 .DEB 格式，很容易安装到基于 Ubuntu 和 Debian 的发行版上。<a href="https://zh.wikipedia.org/wiki/WordPerfect">WordPerfect</a> 诞生于 1980 年，最早是为 Data General 计算机开发的，1983 年发布 DOS 版本，1991 年发布 Windows 版本。
</div>
