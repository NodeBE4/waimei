[Fish Shell 4.0 释出](https://www.solidot.org/story?sid=80672)
------

<div>
有 20 年历史的 Unix shell Fish 释出了 4.0 版本。Fish Shell 4.0 的一大变化是用 Rust 重写了核心 C++ 代码，意味着依赖关系和构建方式的巨变，但这对终端用户没有直接影响。其它变化包括：新的 bind 键符号，相同种子将生成与之前版本不同的随机值，Ctrl-C 将默认清除命令行，Windows Subsystem for Linux 调整，Fish 能作为自安装二进制文件构建，等等。<p></p>
</div>
