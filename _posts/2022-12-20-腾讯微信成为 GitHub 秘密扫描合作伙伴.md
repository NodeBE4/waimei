[腾讯微信成为 GitHub 秘密扫描合作伙伴](https://www.solidot.org/story?sid=73708)
------

<div>
GitHub 官方博客宣布腾讯微信成为其秘密扫描项目的合作伙伴。不是腾讯微信会秘密扫描 GitHub 的所有代码库，而是 GitHub 有一个项目叫秘密扫描（secret scanning），旨在防止开发者的私有令牌对外泄露。开发者在公开项目中硬编码安全凭证日益成为一个严重的安全隐患，秘密扫描就是在发现这些机密信息后警告开发者。与微信合作意味着 GitHub 将在公开代码库中扫描微信官方帐户和小程序开发者相关的私有令牌，发现之后允许微信撤销这些暴露在外的令牌。<p></p>
</div>
