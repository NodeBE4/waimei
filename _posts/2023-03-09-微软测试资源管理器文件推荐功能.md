[微软测试资源管理器文件推荐功能](https://www.solidot.org/story?sid=74343)
------

<div>
微软开始测试名为“文件资源管理器”的“文件推荐”功能，如名所示，当用户访问文件资源管理器主标签，它会在顶部展示你可能想要打开的特定文件。该功能目前只提供给 Windows 11 Insider Preview Build 23403 的部分测试者，且需要登陆 Azure Active Directory 账户。这意味着目前该功能针对的是企业用户。微软表示将会先收集反馈，然后决定如何推送给所有用户。软件巨人也认识到不是所有人都喜欢资源管理器里面出现意想不到的结果。<p></p>
</div>
