[Top-500 npm 包维护者被要求启用 2FA](https://www.solidot.org/story?sid=71704)
------

<div>
GitHub <a href="https://github.blog/changelog/2022-05-31-top-500-npm-package-maintainers-now-require-2fa/">宣布</a> Top-500 npm 包维护者被要求启用 2FA（双因素身份验证）。npm 的生态系统比大多数人想象的更相互交织。此前的研究<a href="https://www.solidot.org/story?sid=62277">发现</a>平均一个 npm 包加载了 39 名不同维护者的 79 个第三方包。研究人员发现，只要入侵 20 名最有影响力的包维护者帐号就可能危及半数 NPM 生态系统。
