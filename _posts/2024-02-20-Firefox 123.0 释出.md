[Firefox 123.0 释出](https://www.solidot.org/story?sid=77398)
------

<div>
Mozilla 释出了 Firefox v123 版本，主要新特性包括：本地网页翻译支持工具提示和表单控件中显示的文本；地址栏设置位于 Firefox 设置的搜索部分；Network Monitor 支持使用上下文菜单 Save Response As 将响应体保存到磁盘；SVG 渐变支持线性 RGB 插值；支持声明式 ShadowDOM；Web Authentication API 支持跨域凭证创建，等等。<p></p>
</div>
