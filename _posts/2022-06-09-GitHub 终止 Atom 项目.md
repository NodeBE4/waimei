[GitHub 终止 Atom 项目](https://www.solidot.org/story?sid=71776)
------

<div>
GitHub <a href="https://github.blog/2022-06-08-sunsetting-atom/">正式宣布</a>关闭文本编辑器项目 Atom，所有代码库将在 12 月 15 日归档。项目创始人 Max Brunsfeld 则<a href="https://news.ycombinator.com/item?id=31668426">推出了</a> Atom 精神续作——用 Rust 语言开发的 <a href="https://zed.dev/" target="_blank">Zed</a>。GitHub 是在 2011 年宣布 Atom，2015 年 5 月发布了 1.0 版本，它吸引了数百万开发者使用。微软在 2015 年推出了自己的文本编辑器 VS Code，2018 年收购 GitHub 后 Atom 成为了微软的项目，过去几年 VS Code 的用户数在不断增长，而 Atom 的使用者逐渐减少。Atom 死亡了，但它的 Electron 框架继续被数以千计的应用广泛使用，其中包括 VS Code、Slack 和 GitHub Desktop。
</div>
