<p>中美洲国家尼加拉瓜上月9日在台湾参加美国举办的民主峰会之际宣布与台断交，并在隔日与北京复交，上演了一场外交角力大戏。</p><p>时隔近一个月，这场大戏竟然还有续集。台湾驻尼加拉瓜前大使吴进木夫妇被爆在台尼断交隔日，获得尼国国籍并滞留当地至今，在台湾被质疑“叛台”。</p><p>台湾外交部在回应此事时，仿佛进入信息盲区，更让吴进木的“叛台”疑云多了几分悬疑。</p><section id="imu"><div id="dfp-ad-imu1">        </div></section><p><strong>吴进木在断交前夕办退休</strong></p><p>民进党前立委林浊水上周日（2日）在面簿开炮，批评吴进木在台尼断交前夕办退休，且一断交就得到尼国国藉。而断交前夕报回来的都是好消息，发生断交后却不返台报告，直言吴进木一连串行为都不合理。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/2022-01/wujinmu_ziyoushibao.jpg?VersionId=yNg9EJWymrdDxmgZGIgRdTe1zfvmHdcX" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>两度出任台湾驻尼加拉瓜大使的吴进木被指在台尼断交前夕办退休，且一断交就得到尼国国藉，被疑“叛台”。（自由时报资料照）</figcaption></figure>                  <p>台湾外交部发言人欧江安隔天回应说，吴进木去年9月28日因个人生涯规划申请退休获准，已在11月17日办妥相关手续离任。</p><div id="innity-in-post"></div><div id="dfp-ad-midarticlespecial">        </div><p>欧江安还称，因吴进木太太的健康原因，目前暂时无法返回台湾，外交部长吴钊燮已在上月23日在立法院应询时提到吴进木因为“妻子身体不佳，无法长途旅行，已申请退休”，外交部目前没有进一步评论。</p><p>不过，有关吴进木在断交后获尼国国籍的情况，台湾外交部承认事前并不知情。</p><p>针对吴进木以妻子身体为由不返台的解释，中时新闻网引述当地侨界人士反驳称，吴夫人直到日前都还有参与当地活动，外表看不出异状。</p><p>林浊水周二（4日）再度质疑，（即使）太太不适坐飞机，遇到这（断交）在大使来说的天大地大的事，不该自己飞回来？</p><p>外交部在当天的线上例行记者会上再次回应称，已透过私人管道联系吴进木，吁请他尽快针对外界质疑进行说明，并称势必要查明，是否吴进木早已瞒着外交部申请入籍，但外交部却始终被蒙在鼓里。</p><p>这似乎在暗示，外交部对吴进木是否“叛台”存有疑虑，而且在掌握吴进木的动向上，外交部已陷入被动。</p><p>而据尼加拉瓜移民局公文，当局在台尼断交前已在处理吴进木入籍事宜，显示吴早在断交前就已提交申请，尼国顺势在断交隔日核准。</p><p><strong>吴进木被爆在尼加拉瓜置产</strong></p><p>被指“叛台”的疑云未解，台湾媒体这两天又爆料吴进木在尼加拉瓜置产，给“叛台”的传闻再添“证据”。</p><p>根据台媒《联合报》、中央社等报道，台湾监察院财产申报资料显示，吴进木夫妇除在桃园拥有房地产，在美国弗吉尼亚州也有房产，2016年申报的一笔房产，地点疑似在尼加拉瓜。</p><p>吴进木在2020年申报的总存款，折合新台币约为2513万1636元（约124万新元），但其中仅16万4835元为新台币，其余大多是美元。</p><p>有台湾网民质疑，台湾用于“金援外交”的钱是否进了吴进木的口袋，痛批外事部门在“养米虫混日子”。也有网民讽刺，吴进木或对民进党政府“心寒透顶”，提早置产，布局入籍尼加拉瓜。</p><p>厦门大学台研院副院长张文生接受亲北京的港媒《大公报》采访时分析，吴进木担任台湾驻外代表，虽然本身收入就不低，但他手上实际掌握许多经费，包括用金钱为邦交开路的经费，驻外机构远在天边，台湾政府很难掌控。</p><p><strong>两度担任尼加拉瓜大使</strong></p><p>吴进木到底是什么来头？他与尼加拉瓜又有何渊源？</p><p>这名台湾资深外交官首次出任尼国大使是在2007年6月至2011年6月，他之后被派往秘鲁担任驻秘鲁代表，2017年5月又回到尼加拉瓜，二度出任驻尼国大使，前前后后在尼加拉瓜派驻八年多。</p><p>吴进木还被尼加拉瓜授予象征外交官最高荣誉的大十字勋章，以及尼加拉瓜首都马纳瓜市荣誉市民。</p><p>台湾外交圈人士披露，吴进木在首任驻尼加拉瓜大使期间，就和当时第二次担任尼国总统的奥尔特加（Daniel Ortega）结下深厚交情。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/2022-01/daniel_ortega_lutou.jpg?VersionId=qJCm51226dgX4vMtqXtqliGYlYPN7l_9" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>吴进木与尼国总统奥尔特加（如图）交情颇深。（路透社）</figcaption></figure>                  <p>一种说法是，2017年时任台湾外交部长李大维出访中南美洲时，奥尔特加特别指定要吴进木担任尼国大使，这才促成吴二度驻尼。</p><p>另一种说法是，吴进木早在2016年就已办理届龄退休，但刚上任的台湾总统蔡英文苦于中美洲缺乏人手，李大维在外交部到处找人派往尼国，最后找上在尼国深耕多年且与奥尔特加私交甚笃的吴进木。</p><p>台湾前资深外交官介文汲接受台媒TVBS访问时则称，吴进木五年前就已经要退休了，硬是被蔡政府请出来，所以他为台湾扛了五年的邦交。</p><p><strong>与奥尔特加交情深厚</strong></p><p>无论吴进木为何二度出任尼国大使，他与奥尔特加的交情都是可圈可点的一笔。</p><p>消息人士说，吴进木在尼国相当高调，非常喜欢炫耀他和奥尔特加的兄弟情，还对外称奥尔特加常常前往他的官邸下厨餐叙。</p><p>吴进木与奥尔特加夫人兼副总统穆里略（Rosario Murillo）关系也相当不错，他甚至能自由出入奥尔特加夫妇的官邸。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/2022-01/nifufu_lutoushe.jpg?VersionId=hxsGz4qNRmWBAqg.Nm3zjDsdE2AqFeUo" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>吴进木与奥尔特加夫人兼副总统穆里略（左）关系也相当不错，他能自由出入奥尔特加夫妇的官邸。（路透社）</figcaption></figure>                  <p>另有外交人士透露，吴进木在当地很善于推销自己，在他生日时，大使馆附近街道上还会挂满“祝大使生日快乐”的红布条，使馆人员及当地人都相当讶异。</p><p><strong>吴进木外交风骨遭批</strong></p><p>吴进木风波的最大“疑云”，在于他入籍尼加拉瓜和台尼断交在时间上的巧合。</p><p>有台湾舆论质疑，尽管断交之时在任大使已不是吴进木，但他离任不到一个月即发生断交，吴进木此前是否完全不掌握情况？</p><p>也有批评声音指，断交是难以扭转的大势，但作为大使在断交隔天加入尼国国籍，有失外交人员风骨。</p><p>台湾前外交官刘仕杰认为，“叛国罪”罪名兹事体大，不能在没有确切证据的前提下妄下评论，但“整起事件不妥”。他说，作为大使，却没办法守住邦交，理应有责。纵使形势不在我，断交实难扭转，但作为大使，负起责任，并不为过。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/2022-01/liushijie_ziyoushibaodanganzhaoc.jpg?VersionId=UzcsS9nQ5CJa0hyDQyzUV8mxpHBmU1mf" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>台湾前外交官刘仕杰认为，作为大使，却没办法守住邦交，理应有责。（自由时报档案照）</figcaption></figure>                  <p>民进党立委赵天麟则称，成为尼加拉瓜公民是吴进木的自由，但因其退休、成为尼国公民的时间，都距离断交时间很近，其中牵扯到个人自由与台湾利益的竞合，使外界有许多猜测联想。</p><p>吴进木目前尚未返台，民进党立委王定宇认为，若滞留尼加拉瓜不归，除了显示毫无职业道德，也会遭人质疑在职期间是否尽力维护台湾利益，“这样的外交官，一辈子会让人瞧不起。”</p><p>不过，同情的声音则认为，吴进木早已可以退休，但为了尼加拉瓜的邦交，由当时的外交部长李大维再次派往尼国，等于是帮民进党又扛住了五年邦交，否则早已断交了。</p><p>台湾前驻海地大使杨承达告诉中央社，据他了解，吴进木早有在尼国定居的打算，只是没想到会在此时断交。言下之意，几个时间点接近只是巧合，而非吴进木一早预谋，或是串谋里应外合。</p><p><strong>吴进木风波烧到吴钊燮</strong></p><p>吴进木风波发生后，舆论也把矛头对准台湾外交和国安系统，认为无论吴进木是否“叛台”，这些部门都难辞其咎。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/2022-01/joseph_wu_lutou.jpg?VersionId=qZBYEOW_tol97zTQMzTl6xjAgo4VdyfX" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>台湾《联合报》今天在一篇社论中指，整起事件透露出台湾外交部的尴尬，而作为部长的吴钊燮难辞其咎。（路透社）</figcaption></figure>                  <p>林浊水接连发文痛吴进木，并质问“国安系统在做什么？”</p><p>他也质疑，“外交部先说太太不良于行，被拍到很良于行的照片，外交部改口太太不适坐长途飞机，两个理由为何差那么多”，暗指外交部有替吴进木打掩护，同时替自己推责之嫌。</p><p>《联合报》今天在一篇社论中指，整起事件透露出台湾外交部的尴尬。文章写到，有“叛国”或“不忠”之嫌，首先须追讨其责任的应该是外交部长吴钊燮才对，但为何外交部却沉默无言，甚至还帮吴进木编织理由？</p><p>就连亲绿台媒《自由时报》今天刊登的读者投稿，也以《尽速说明吴进木案 外交部有责》为题，向外交部发难。</p><p>实际上，除了责任归属外，吴进木引发的“叛台”风波还有一个更大的潜在风险。</p><p>根据台湾规定，驻外大使为法定涉密人员，应受机密保护法规范，三年之内获核准始得出境，但并未硬性规定要返台述职。民进党立委林淑芬就担忧，大使离职或退休不返台，政府要如何掌握涉密人员在外的活动行为？</p><p>对此，吴钊燮在上月应询时说，他已提醒吴进木本人受到台湾机密保护法规范，未来会检讨这部分可能的漏洞，让相关规范更为严谨。</p><p>台湾外交部面对外交官“叛台”甚至泄露“国家机密”的风险，却滞后地给出一个软绵绵的空洞回应，难怪在野党国民党副主席连胜文会讽刺蔡英文大部分时间都花在公投“洗脑”老百姓，对于邦交国状况都不知道。如此一来，今后再出现第二个吴进木或许不足为奇。</p>
