[律师宣布调查 GitHub Copilot 版权侵犯问题](https://www.solidot.org/story?sid=73117)
------

<div>
GitHub 的 AI 编程助手 Copilot 使用公开的代码库进行训练，在开发者写代码时根据函数名等上下文自动补充完后续代码。很多时候 Copilot 补充的是公开代码库中代码片段的拷贝。这就引起了版权相关的问题。律师 Matthew Butterick 本周宣布与律所 Joseph Saveri 合作<a href="https://www.theregister.com/2022/10/19/github_copilot_copyright/" target="_blank">调查对 Copilot 提起版权索赔的可能性</a>，包括：GitHub 用开源代码训练 Copilot 是否有不当行为；在补充代码时是否不当的推荐了其他人的版权作品。自由软件基金会（FSF）此前曾<a href="https://www.solidot.org/story?sid=68448">呼吁</a>对 GitHub Copilot 所产生代码的许可授权问题展开调查。
</div>
