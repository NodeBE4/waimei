[微软将 Linux sudo 命令带到 Windows Server](https://www.solidot.org/story?sid=77309)
------

<div>
最新泄露的微软 Windows Server 2025 Insider 预览版被发现包含了 Windows sudo 命令的新设置。相关设置需要启用开发者模式，且 sudo 命令在命令行中无法运行，显示它仍然处于开发初期。不过它的设置提供了其如何工作的线索，它允许在“新窗口”、“禁用输入”和“内联（Inline）”情况下运行 sudo 应用。Windows 11 尚未发现 sudo 命令，但不排除会在未来的更新中加入。值得一提的是，微软在预览版本中测试的新功能不一定会进入正式的生产版本。<p></p>
</div>
