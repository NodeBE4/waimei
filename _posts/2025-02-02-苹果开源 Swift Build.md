[苹果开源 Swift Build](https://www.solidot.org/story?sid=80461)
------

<div>
苹果宣布开源 Swift Build，源代码托管在 GitHub 上，采用 Apache License 2.0 许可证。Swift 语言被用于为苹果旗下的各种操作系统开发软件，Swift Build 是一个强大且可扩展的构建引擎，提供了一套用于构建 Swift 项目的构建规则。Swift Build 是 Xcode 使用的引擎，开源库还包括对 Linux 和 Windows 的支持。<p></p>
</div>
