[Twitter 指微软不当使用其 API](https://www.solidot.org/story?sid=75005)
------

<div>
Twitter 致函微软，称后者“似乎将 Twitter API 用在了未经授权的用途和目的上”。Twitter 的老板马斯克(Elon Musk)此前曾多次公开指责这家软件巨头。律师在这封信中说：“微软可能在很长一段时间里违反了协议中的多项规定，”信中指出，多年来，微软在部分产品中免费使用 Twitter 的标准开发者 API。微软上个月拒绝为继续使 用Twitter API 付费。Twitter 要求微软披露过去两年使用 Twitter API 的相关情况，包括微软目前掌握哪些 Twitter 内容，以及微软是如何使用 Twitter API 的。微软的一位代表证实该公司已收到这封信函，表示将进行评估并作出适当答复。<p></p>
</div>
