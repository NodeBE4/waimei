[一名 Rust Linux 维护者辞职](https://www.solidot.org/story?sid=79109)
------

<div>
Rust for Linux 内核维护者 Wedson Almeida Filho 宣布了辞职，导致他辞职的一大原因是围绕内核使用 Rust 语言的非技术性争论。Wedson 是微软工程师，参与了大量与 Rust Linux 内核功能相关的工作，包括实验性的将 EXT2 文件系统驱动移植到 Rust。但现在他宣布自己受够了，表示自己没有了回应非技术性争论的精力和热情，他更热衷于与 Rust for Linux 团队讨论技术性问题，坚信内核的未来是采用内存安全的语言。<p></p>
</div>
