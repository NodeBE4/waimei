[Google、GitHub 和 Azure 冻结招聘](https://www.solidot.org/story?sid=72215)
------

<div>
因不利的经济环境 Google、GitHub 和 Azure <a href="https://www.linkedin.com/posts/gergelyorosz_hiring-softwareengineering-google-activity-6955782106832592896-MPnL">冻结招聘</a>，立即生效。GitHub 将继续招聘战略/关键职位，但普通职位招聘已经关闭。Google 冻结招聘两周，预计将重新分配员工。Azure 需要副总裁级别的高管批准才能增加员工。Meta 的软件工程师招聘自 3 月以来就已经冻结，Twitter 的情况类似。大型科技公司冻结招聘将会对整个软件行业产生影响，因为它们是抬高薪酬的主要因素。
</div>
