[WebKit 迁移到 Git 和 GitHub](https://www.solidot.org/story?sid=72638)
------

<div>
WebKit 项目在 6 月 23 日<a href="https://webkit.org/blog/13140/webkit-on-github/">冻结了</a> Subversion 树，迁移到<a href="https://github.com/WebKit/WebKit" target="_blank">托管</a>在 GitHub 的 Git 源代码管理系统。Git 系统简单便捷，开发者已对其十分熟悉，在迁移之前 WebKit 开发者就更倾向于在 git-svn 镜像上工作。选择托管在 GitHub 是因为它有庞大的 Web 开发者社区，他们能与 WebKit 项目更紧密的合作去改进浏览器引擎。
</div>
