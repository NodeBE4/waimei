[美国参议员在 GitHub 上公布加密货币监管法案](https://www.solidot.org/story?sid=71942)
------

<div>
美国参议员 Cynthia Lummis（R-WY）在 GitHub 上公布了准备提出的加密货币监管法案 <a href="https://github.com/responsible-financial-innovation-act22/RFIA-bill">Responsible Financial Innovation Act</a>，目前吸引了 56 个 Issues 和 21 个 Pull requests，其中很多是恶作剧性质的。也有人在认真讨论，认为作者需要遵循 GitHub 上发布内容的模式。还有人提议对工作量证明（Proof of Work，PoW）加密货币征收挖矿税。工作量证明货币如比特币在挖矿过程中需要消耗大量的能量，因此引发了很多争议。GitHub 采用的 Git 版本控制系统可以跟踪所有变化，有人<a href="https://slashdot.org/comments.pl?sid=21603058&amp;cid=62648858">提议</a>在法律制定和法案起草过程中强制使用 Git，能让人们更容易的了解变化。
</div>
