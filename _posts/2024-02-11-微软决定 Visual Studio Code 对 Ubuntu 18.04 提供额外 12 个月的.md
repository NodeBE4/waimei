[微软决定 Visual Studio Code 对 Ubuntu 18.04 提供额外 12 个月的支持时间](https://www.solidot.org/story?sid=77345)
------

<div>
微软最近释出的 Visual Studio Code 更新（v1.86 或 2024 年 1 月更新）影响到了 Ubuntu 18.04 LTS 或其它使用 RHEL 7 和 CentOS 7 等发行版的开发者，原因是微软将 Visual Studio Code 远程开发工具的最低构建版本升级到了 glibc 2.28，而 Ubuntu 18.04 LTS 等发行版使用的版本是 glibc 2.27，这导致了编辑器停止工作。Canonical 为 Ubuntu 18.04 LTS 提供了 10 年支持，一直支持到 2028 年，升级一个长期支持版本的重要软件包是不太可能的。在引发广泛争议之后，微软决定对 Ubuntu 18.04 LTS 等发行版给予“缓刑”，给这些发行版用户额外的 12 个月时间。微软 VS Code 高级产品经理 Isidor Nikolic 表示希望开发者及其公司能在 12 个月内迁移到新的发行版。<p></p>
</div>
