[微软解决导致 Edge 停止响应的 bug](https://www.solidot.org/story?sid=71963)
------

<div>
微软<a href="https://docs.microsoft.com/en-us/windows/release-health/status-windows-11-21h2#ie-mode-tabs-in-microsoft-edge-might-stop-responding">解决了</a>导致运行在 IE 模式下的 MS Edge 浏览器停止响应的 bug。该问题影响安装了可选累计更新 <a href="https://support.microsoft.com/help/5014019" target="_blank" rel="nofollow noopener">KB5014019</a>、<a href="https://support.microsoft.com/help/5014023" target="_blank" rel="nofollow noopener">KB5014023</a> 和 <a href="https://support.microsoft.com/help/5014021" target="_blank" rel="nofollow noopener">KB5014021</a> 的 Windows 11 (version 21H2)、Windows 10 (versions 20H2、221H1 和 21H2 以及 Windows Server 2022，在安装上述更新之后， 当一个网站展示模态对话框后 Microsoft Edge 的 IE 模式标签页将停止响应。微软解释说，所谓的模态对话框是指一种表格或对话框，要求用户在与网站继续互动前做出回应。微软通过已知问题回滚（<a href="https://www.bleepingcomputer.com/news/microsoft/microsoft-windows-10-known-issue-rollback-auto-fixes-update-bugs/" target="_blank">Known Issue Rollback</a>）修复了该问题。
</div>
