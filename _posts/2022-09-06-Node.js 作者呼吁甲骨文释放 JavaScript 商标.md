[Node.js 作者呼吁甲骨文释放 JavaScript 商标](https://www.solidot.org/story?sid=72695)
------

<div>
Node.js 和 Deno 的作者 Ryan Dahl 发表公开信<a href="https://devclass.com/2022/09/05/node-js-creator-ryan-dahl-urges-oracle-to-release-javascript-trademark/?td=rt-3a">督促</a>甲骨文将 JavaScript 商标释放到公有领域。JavaScript 的名字长期以来一直让人困惑，人们以为它与 Java 相关，但实际上除了都是花括号语言外两者毫无相似之处。当微软为 IE 创造一种 JS 实现时它将其称为 JScript，之后又创造出一种 .NET 实现。Netscape 与微软等公司合作标准化 JS 语言，选择了一个中立的名词 ECMAScript。但用 JS 作者 Brendan Eich 的话说，ECMAScript 的名字不招人喜欢，听起来像是一种皮肤病。当甲骨文在 2010 年收购 Java 时它也拥有了 JavaScript 商标。也就是只有甲骨文公司可以允许一种语言叫 JavaScript。甲骨文本身没有任何使用 JavaScript 商标的产品。
</div>
