[新 D12 到 Metal 翻译将 Windows 游戏带到 macOS 上](https://www.solidot.org/story?sid=75165)
------

<div>
苹果的 macOS 系统不是理想游戏平台。为了能让 Windows 游戏能在 macOS 上运行，最理想的方案是将微软的 DirectX 12 API 翻译到苹果的 Metal API，类似基于 Linux 的 Steam Deck 的做法。 CodeWeavers 的 CrossOver 软件将初步支持此类的软件兼容层，它发布了在 Apple M2 上运行《Diablo II Resurrected》的屏幕截图。CodeWeavers 开发者表示，将 DirectX 12 API 翻译到 Metal API 可能需要对每一个游戏进行针对性的修正，因此在 macOS 上运行 Windows 游戏达不到 Steam Deck 的那种游戏兼容性水平。<p></p>
</div>
