<p>行事作风大胆、常常语出惊人的美国电动汽车巨头特斯拉创始人埃隆·马斯克今天上午分别在推特和微博上以一首中国古诗《七步诗》再次轰动全网。</p><p>这条全文为：“Humankind，煮豆燃豆萁，豆在釜中泣，本是同根生，相煎何太急”的推文两小时即获得五万人点赞、7000多人转发。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/2021-11/tuite.jpg?VersionId=v8uvRtDpqWOXfKaJwRKPMldTt7drEKyw" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>马斯克在推特上以中文发中国古诗，让许多国外网民一头雾水。（推特）</figcaption></figure>                  <p>就在同天出炉的彭博全球亿万富豪排行榜上，全球首富马斯克的个人财富随着特斯拉股价的持续上涨而水涨船高至3350亿美元，已是著名投资人巴菲特的三倍。结合马斯克近期动向，他贴出这首出自北魏诗人、也是曹操之子的曹植讽刺骨肉相残的诗词，随即引发网民大猜想。</p><section id="imu"><div id="dfp-ad-imu1">        </div></section><p><strong>《七步诗》的几大猜想</strong></p><p>自10月下旬以来，美国议员和国际组织都曾发声要求马斯克拿出部分个人财富。联合国世界粮食计划署负责人比斯利（David Beasley)上周则呼吁超级富豪帮助解决世界饥饿问题，并公开点名马斯克。</p><p>马斯克随后在推特上回应称，“如果联合国能证明60亿美元可以解决世界饥饿问题，他将马上出售手中的特斯拉股票。”</p><div id="innity-in-post"></div><div id="dfp-ad-midarticlespecial">        </div><p>因此许多网民认为，马斯克此时贴出《七步诗》是在回应上述事件。有网民调侃说，比起《七步诗》，描述农民辛苦得来不易的“锄禾日当午”似乎更适合。</p><p>而作为加密货币的坚定支持者，不少网民则猜测马斯克贴出《七步诗》是暗指近期火热的狗狗币（Dogecoin）和柴犬币（Shiba Inu）之间的争端，并解读该七步诗的内容实则为“煮doge燃doge萁，doge在釜中泣。”</p><p>有人认为，马斯克提及的“Humankind”是一款以人类文明历史发展为题材的策略游戏，他可能只是在单纯的抒发“玩后感”；也有网民称“两岸局势连马斯克都知道”，还有人在四句诗后又续了两句“地球容不下，我走行不行”。</p><p>对于推特上不识中文的吃瓜群众来说，他们在使用翻译软件后坚定的认为，马斯克是在分享一道与豆子有关的菜谱：“Boiling beans by burning the beanstalks”。</p><p>江湖传闻带有中国血统的马斯克此次发中国古诗为网络带来不少谈资，但也并不让人意外。自特斯拉2013年正式进军中国以来，马斯克就与中国结下不解之缘。</p><p>从2014年几乎年年到访中国的马斯克每次都对中国的方方面面赞不绝口，就连街边的包子铺、火锅店、煎饼摊也没放过，这使得有“硅谷钢铁侠”之称的狂人马斯克成功打造出“中国迷弟”人设。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/2021-11/baozi.jpg?VersionId=VlrdKCmIUWo7CSb0lwCoPvx04KHqb4.Q" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>马斯克2019年在北京一家包子铺用餐引众人围观，随后发推特赞“好吃”。（互联网）</figcaption></figure>                  <p>马斯克2014年首次访华时就大赞中国的高效；2015年面对中国经济放缓趋势，马斯克又充满信心的高调喊话：“中国现在、未来都非常有吸引力”。</p><p>2018年，马斯克在看完一则《中国工人在9个小时内建造完成一个火车站》的视频后，在推特上转发并狂赞：中国基建效率是美国的一百倍！</p><p>同年7月，马斯克与上海方面签订协议，在地区独资建设超级工厂，使得特斯拉成为中国首家外资全资拥有的汽车工厂。五个月后，马斯克在中南海紫光阁受到中国国务院总理李克强接见，迎来他在中国高歌猛进的“高光时刻”。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/2021-11/2018niantiaowu_masike.jpg?VersionId=jfiGmIDWTTG5.RRtu5r51EmiPEnsT_kO" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>马斯克在2019年特斯拉上海工厂奠基仪式上手舞足蹈。（互联网）</figcaption></figure><figure class="image-placeholder "><img alt class <figcaption>马斯克2019年1月在中南海紫光阁受到中国国务院总理李克强接见。（路透社）</figcaption></figure>                  <p>不难看出马斯克和中国难解之缘，也源于这个市场的巨大潜能。今年前三季度，特斯拉在中国实现营收90.15亿美元，同比增加122.92%。中国市场营收在全球占比24.97%，去年同期，这一数字则为19.45%。</p><p>马斯克今年3月接受中国央视采访时就预言：“长远来看，中国将成为特斯拉的最大市场。”</p><p>然而，一路顺风顺水的马斯克和特斯拉从去年开始却波澜不断，这个被视为“中国人民的老朋友”的马斯克开始被中国舆论指责“态度傲慢”、“被胜利冲昏头脑”。</p><p>去年3月，特斯拉因Model 3车型部分车辆违规装配HW2.5组件问题，被中国工业和信息化部装备工业一司约谈，并责令其按照有关规定立即整改，切实履行企业主体责任；</p><p>今年2月，特斯拉召回36126辆Model S和Model X，一举超越其2020年召回总量，同时，特斯拉还因异常加速、电池起火、车辆远程升级等问题再次被有关部门约谈；</p><p>今年4月，上海车展发生的“维权事件”更是让特斯拉在中国的声誉跌至低谷。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/2021-11/chezhan0.jpg?VersionId=lnw0AqZKT4tpo5kNAa.5Ws_U_78Vr9TP" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>今年4月上海车展上，一名女子身穿印有“刹车失灵”和“隐形杀手”字样的T恤，站在一辆红色的特斯拉车顶，不断咆哮着“特斯拉刹车失灵”。（互联网）</figcaption></figure>                  <p>而随着中美两国战略竞争的加剧，消息人士透露，中国军方今年3月以安全隐患为由，禁止特斯拉汽车进入军事设施区域。 两个月后，再传出由于特斯拉车内摄像头存在安全隐患，中国部分政府机关的工作人员被告知不要将他们的特斯拉汽车停放在政府办公区域内。至此，特斯拉的中国之旅可谓岌岌可危。</p><p><strong>马斯克的力挽狂澜</strong></p><p>经过了蜜月期，又面临着七年之痒，马斯克如何面对与中国的风风雨雨分分合合呢？</p><p>实际上，马斯克今年一系列的高调挺华言论和伴随而来的特斯拉在华销量反弹都说明，他正渐渐走出这场危机。</p><p>在特斯拉卷入数据安全风波后，马斯克随即在由中国国务院下属基金会主办的中国发展高层论坛的虚拟讨论中回应说：如果特斯拉汽车用于间谍活动，他的公司将被关闭。</p><p>三天后，马斯克在接受中国央视采访时表示，中国最新五年经济和社会蓝图中设定的碳排放目标给他留下了深刻印象，并称赞“中国正朝着成为世界最大经济体的方向发展，未来更加繁荣昌盛。”</p><p>特斯拉官方微博今年5月25日发布消息称，特斯拉已经在中国建立数据中心，以实现数据存储本地化，并将陆续增加更多本地数据中心。所有在中国大陆市场销售车辆所产生的数据，都将存储在境内。</p><p>今年9月在乌镇举行的世界互联网大会上，马斯克在开幕式发表致辞时重申了上述消息，并预言中国将成为数字化的全球领导者，而特斯拉将继续加大在中国的投资和研发力度。</p><p>另一方面，就在上海车展“维权事件”事件隔日，马斯克的母亲梅耶·马斯克以模特兼营养师的身份低调入驻了中国热门社交平台——小红书。这个虽已73岁高龄但仍然风姿绰约的时髦老太太以日常工作的视频吸引了许多粉丝关注，这也是梅耶·马斯克继新浪微博后在中国开通的第二个社交平台。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/2021-11/xiaohongshu.jpg?VersionId=Ro98ZffsCqmgtHfumpejgxvFpmmM_Wr5" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>马斯克的母亲梅耶·马斯克今年四月入驻了中国热门社交平台——小红书，被指是为马斯克危机公关。（小红书截取）</figcaption></figure>                  <p>她在第一则视频中用英文介绍自己，并称去过中国几次，很喜欢这个国家。尽管此举被一些网民指是为了给特斯拉做危机公关，但总体上仍得到不少好评。</p><p>与此同时，作为加密货币的最大拥趸，马斯克今年3月曾宣布旗下电动车公司特斯拉将支持比特币支付，促使比特币在4月走上历史高位。但到了5月，他忽然“反悔”，称已暂停使用比特币购买特斯拉的计划，给出的理由是环保，比特币也因此应声下跌10%。</p><p>值得注意的是，中国政府同月为打击加密货币出台了一系列政策，包括金融和支付机构不得接受加密货币作为支付结算工具；不得提供与加密货币相关的服务和产品；打击比特币挖矿和交易行为等。从时间线来看，马斯克的表态与中国政府的政策或许不只是一种巧合。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/2021-11/biter.jpg?VersionId=yNbHq6QhNHmROa.PNYXLEeEvuiTjxWGJ" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>马斯克今年三月曾宣布旗下电动车公司特斯拉将支持比特币支付，但两个月后，他又“反悔”宣布暂停使用比特币购买特斯拉的计划。（路透社）</figcaption></figure>                  <p>尽管比特币市场因为马斯克的言论反复震荡，但特斯拉在中国的形象和销量却在稳步提升。</p><p>特斯拉的相关负面新闻在这一时期悄悄“反转”。先是6月初网上一段录音被曝光，显示此前投诉特斯位“刹车失灵”的郑州温先生自称自己的车子并无大碍，只是希望利用舆论压力让特斯拉对他“一赔三”。连同一直连续报道温先生“维权”的河南电视台的微博账号“河南广播电视台”等也删除了相关内容。</p><p>随后中国官媒《人民日报》旗下新媒体平台“人民车市”6月底发文，力证特斯拉的刹车很正常。文章点名批评上海车展“维权”女士，称“适度维权可以，但不要心存恶念、断章取义、颠倒黑白、存司马昭之心”。</p><p>“812温州特斯拉失灵”车主名誉权纠纷一案也于上月作出判决，认定被告车主发布与事实不符、损害特斯拉声誉的信息，造成网络用户、媒体观众等对特斯拉的错误认知，以及对特斯拉社会评价降低，责成被告向原告特斯拉赔礼道歉，并支付赔偿款5万元（人民币，下同，1万零515新元）。</p><p>而就在乘联会上月12日发布的中国9月新能源汽车的销量情况中，特斯拉再次刷新了自己创下的销量纪录。数据显示，特斯拉9月共计售出了5万6006辆新车，打破了8月超4万辆的表现，环比增长27%，同比增长394%。</p><p>在这5万6006辆新车中，有5万2153辆均交付于中国本土车主，显示特斯拉在国内经历低谷后再次赢回市场信任。</p><p>今天马斯克以一首中国古诗引发各方猜想，未来马斯克在中国的发展也备受各方关注，马斯克面对的究竟是中国结还是中国劫，他又会怎样应对，值得拭目以待。</p>
