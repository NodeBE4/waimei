[Firefox 105.0 释出](https://www.solidot.org/story?sid=72833)
------

<div>
Mozilla <a href="https://www.mozilla.org/en-US/firefox/105.0/releasenotes/">释出</a>了 Firefox 105.0。主要新变化包括：打印预览对话框加入只打印当前页选项；在第三方上下文中支持分割 Service Worker——<a href="https://developer.mozilla.org/en-US/docs/Web/API/Service_Worker_API" target="_blank">Service Worker</a> 充当了 Web 应用、浏览器和网络之间的代理服务器功能，旨在创造一种离线体验，拦截网络请求并根据网络是否可用采取适当行动，更新服务器上的资源；Windows 版本启用滑动导航；兼容 User Timing L3 规格，搜索大列表中个别项目的速度提升一倍；改进 Windows 版本的稳定性，bug 修复等等。
</div>
