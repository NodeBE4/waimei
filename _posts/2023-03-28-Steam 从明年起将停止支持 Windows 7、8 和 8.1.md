[Steam 从明年起将停止支持 Windows 7、8 和 8.1](https://www.solidot.org/story?sid=74514)
------

<div>
Valve 通过其支持文档宣布它的 Steam 客户端从明年 1 月 1 日起将停止支持 Windows 7、8 和 8.1。之后  Steam 客户端将停止在这些操作系统上运行。为了继续运行 Steam 以及运行通过 Steam 购买的游戏，用户需要更新到新版本的 Windows。Valve 解释说，Steam 客户端是基于 Google Chrome 的嵌入式版本，而 Chrome 在旧版本的 Windows 上不再工作。未来版本的 Steam 将需要 Windows 10 及以上版本。Google 今年初发布的 Chrome 109 是最后一个支持 Windows 7 和 Windows 8/8.1 的版本。<p></p>
</div>
