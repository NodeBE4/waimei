[微软 Linux 发行版 CBL-Mariner 发布 2.0 版本](https://www.solidot.org/story?sid=71500)
------

<div>
微软有自己的 Linux 发行版 CBL-Mariner，去年<a href="https://www.solidot.org/story?sid=68310">发布</a>了 1.0 版本，本周发布了<a href="https://github.com/microsoft/CBL-Mariner/releases/tag/2.0.20220426-2.0"> 2.0 版本</a>，源代码公布在 Github 上。CBL-Mariner 不是桌面 Linux 而是服务器端 Linux，它在微软内部被用于云基础设施以及边缘计算产品和服务。它提供了一组云和边缘服务所需的通用软件包集，可以很容易根据需要引入额外的软件包。它是一个轻量级的发行版，可作为容器或容器主机使用。
</div>
