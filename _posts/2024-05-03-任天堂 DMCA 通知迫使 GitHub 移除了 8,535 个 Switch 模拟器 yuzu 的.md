[任天堂 DMCA 通知迫使 GitHub 移除了 8,535 个 Switch 模拟器 yuzu 的副本](https://www.solidot.org/story?sid=78071)
------

<div>
即使开源 Switch 模拟器项目 yuzu 已经因诉讼而关闭，任天堂仍然在追捕网络上的 yuzu 副本或克隆项目。它向最大代码托管平台 GitHub 发出了 DMCA 删除通知，迫使 GitHub 移除了 8,535 个 Switch 模拟器 yuzu 的副本库。任天堂的理由是 yuzu 非法绕过任天堂的技术保护措施以及运行任天堂 Switch 游戏的非法副本。另一个代码托管平台 GitLab 也已经删除了 yuzu 代码库。<br><p></p>
</div>
