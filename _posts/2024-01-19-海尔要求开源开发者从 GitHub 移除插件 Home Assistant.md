[海尔要求开源开发者从 GitHub 移除插件 Home Assistant](https://www.solidot.org/story?sid=77179)
------

<div>
海尔向一位开源开发者发出律师函，要求从 GitHub 上移除其维护的插件 Home Assistant。Home Assistant 是一个开源家庭自动化平台，允许用户从一个集中式的界面控制和自动化智能家居设备。它支持控制海尔、Candy 和 Hoover 的空调、净化器、洗碗机、电磁炉、烤箱、冰箱、洗衣机和烘干机。海尔的律师声称该开源插件给公司造成了重大财务损失，违反了版权法，要求将其删除以免面临法律诉讼。此事引发了部分用户的强烈不满，呼吁抵制海尔的产品。开发者 Andre Basche 表示将在数天内下架插件。<p></p>
</div>
