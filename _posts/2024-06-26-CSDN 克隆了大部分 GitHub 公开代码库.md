[CSDN 克隆了大部分 GitHub 公开代码库](https://www.solidot.org/story?sid=78526)
------

<div>
CSDN 的代码托管平台 GitCode 被发现克隆/镜像了大部分 GitHub 公开的代码库，它甚至还克隆了每个代码库的星标——GitHub 上的一种关注和收藏机制。大部分代码库使用的开源许可证允许自由克隆或镜像，因此 CSDN 此举称不上非法，但被认为缺乏道德。<p></p>
</div>
