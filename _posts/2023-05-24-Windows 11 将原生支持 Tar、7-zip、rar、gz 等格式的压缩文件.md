[Windows 11 将原生支持 Tar、7-zip、rar、gz 等格式的压缩文件](https://www.solidot.org/story?sid=75041)
------

<div>
微软宣布 Windows 11 将原生支持 Tar、7-zip、rar、gz 等格式的压缩文件。以前在 Windows 上打开此类压缩文件需要安装第三方解压缩软件如开源的 7-zip。微软负责 Windows 业务的首席产品官 Panos Panay 表示 Windows 11 即将释出的更新将原生支持使用开源项目 libarchive 的压缩格式，该功能将通过本周晚些时候释出的一个开发版本提供给用户。<p></p>
</div>
