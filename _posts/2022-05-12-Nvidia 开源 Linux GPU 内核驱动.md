[Nvidia 开源 Linux GPU 内核驱动](https://www.solidot.org/story?sid=71503)
------

<div>
Nvidia <a href="https://developer.nvidia.com/blog/nvidia-releases-open-source-gpu-kernel-modules/">宣布</a>开源其 Linux GPU 内核驱动模块，<a href="https://github.com/NVIDIA/open-gpu-kernel-modules">源代码</a>发布在 GitHub 上。Nvidia 承认其内核驱动目前的形式还不适合进入上游，意味着内核主线暂时还无法合并其内核驱动。此外开源的驱动不包含用户空间的部分，Nvidia GPU 在 Linux 平台上仍然需要闭源代码。AMD 在 Linux 平台早就统一了开源和闭源驱动，Nvidia 长期以来一直被批评对 Linux 支持不友好，它的私有驱动表现出色，但开源驱动 Nouveau 项目主要通过逆向工程实现对 Nvidia GPU 的支持，功能上有欠缺。Nvidia 在其新闻稿中表示它计划找到对上游更友好的方法。
</div>
