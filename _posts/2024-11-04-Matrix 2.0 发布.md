[Matrix 2.0 发布](https://www.solidot.org/story?sid=79668)
------

<div>
去中心化联邦架构通信协议 Matrix 宣布了 2.0 版本。主要新特性包括：Simplified Sliding Sync，该 API 被用于即时登录、即时启动和即时同步；下一代身份验证，从自己定制的身份验证 API 迁移到行业标准的 OpenID Connect；端对端加密群组语音和视频会议功能 MatrixRTC；端对端加密无缝且不可见。<p></p>
</div>
