[GitHub 废除基于密码的 Git 身份验证](https://www.solidot.org/story?sid=68537)
------

<div>
最大的代码托管平台 GitHub 在 13 日周五这天<a href="https://github.blog/changelog/2021-08-12-git-password-authentication-is-shutting-down/" target="_blank">正式废除了</a>基于密码的 Git 身份验证。从 09:00 PST 开始，使用 GitHub 开发者将需要切换到基于令牌的身份验证去执行 Git 操作，基于令牌的认证包括个人接入、OAuth、SSH Key 活 GitHub App 安装令牌。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
