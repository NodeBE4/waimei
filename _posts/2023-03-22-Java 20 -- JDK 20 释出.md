[Java 20 / JDK 20 释出](https://www.solidot.org/story?sid=74460)
------

<div>
甲骨文宣布 Java 20 的参考实现 JDK 20 GA（Generally Available）。GPL 授权的 OpenJDK 20 也一同发布。Java 20 / JDK 20 的主要新特性包括：JEP 429：作用域值（孵化）；JEP 432：记录类的模式（第二预览版本）；JEP 433：使用 switch 的模式匹配（第四预览版本）；JEP 434：外部函数与内存 API（第二预览版本）；JEP 436：虚拟线程（第二预览版本）；JEP 437：结构化并发（第二孵化版本）；JEP 438：向量 API(第五孵化版本)。此外还有大量的增强和 bug 修复。<p></p>
</div>
